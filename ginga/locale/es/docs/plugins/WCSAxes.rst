Un complemento para generar una superposición de ejes WCS en la imagen cargada.

**Tipo de complemento: Local**

``WCSAxes`` es un complemento local, lo que significa que está asociado a un
canal.  Se puede abrir una instancia para cada canal.

**Uso**

Siempre que la imagen tenga un WCS válido, se mostrarán los ejes WCS.
Use la interfaz del complemento o el archivo de configuración para personalizar
la visualización de los ejes.
