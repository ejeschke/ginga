El modo Dibujo activa asignaciones de teclas que facilitan el dibujo.

Entrar en el modo mediante
--------------------------
* Espacio, luego «b»

Salir del modo mediante
-----------------------
* Esc

Asignaciones predeterminadas del modo
-------------------------------------
* v : añadir un vértice a un polígono, mientras se dibuja o edita un lienzo
* z : eliminar un vértice de un polígono, mientras se dibuja o edita un lienzo
* x : eliminar un objeto seleccionado mientras se edita un lienzo
* arrastrar con el izquierdo : dibujar en el lienzo con la forma definida
* arrastrar con el derecho : dibujar en el lienzo con la forma definida
