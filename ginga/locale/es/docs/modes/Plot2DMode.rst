El modo Plot2D activa asignaciones de teclas que permiten fijar la posición de
panorámica y el nivel de zoom (escala) en un visor de gráficos de Matplotlib
(módulo ginga.plot.PlotView).

Entrar en el modo mediante
--------------------------
* Espacio, luego «p»

Salir del modo mediante
-----------------------
* Esc

Asignaciones predeterminadas del modo
-------------------------------------
* Mayús + clic izquierdo : fijar la posición de panorámica
* clic central : fijar la posición de panorámica
* desplazar : acercar/alejar
* Ctrl + desplazar : acercar/alejar solo el eje X
* Mayús + desplazar : acercar/alejar solo el eje Y (en MacOS, solo trackpad)
* Alt + desplazar (ratón) : acercar/alejar solo el eje Y (tecla Option en Mac)
* Alt + desplazar : acercar/alejar en el cursor

Asignaciones de teclado
-----------------------

Zoom
----
* igual : acercar un nivel de zoom
* Ctrl + igual : acercar el eje X un nivel de zoom
* más (Mayús + igual) : acercar el eje Y un nivel de zoom
* menos : alejar un nivel de zoom
* Ctrl + menos : alejar el eje X un nivel de zoom
* guion bajo (Mayús + menos) : alejar el eje Y un nivel de zoom
* 9 : alejar manteniendo la posición del cursor
* Ctrl + 9 : alejar el eje X manteniendo la posición del cursor
* paréntesis izquierdo (Mayús + 9) : alejar el eje Y manteniendo la posición del cursor
* 0 : acercar manteniendo la posición del cursor
* Ctrl + 0 : acercar el eje X manteniendo la posición del cursor
* paréntesis derecho (Mayús + 0) : acercar el eje Y manteniendo la posición del cursor
* acento grave : ajustar los ejes X e Y a la ventana
* 1 : ajustar solo el eje X a la ventana
* 2 : ajustar solo el eje Y a la ventana
* k : fijar el límite inferior de X al valor X en el cursor
* l : fijar el límite superior de X al valor X en el cursor
* K : fijar el límite inferior de Y al valor Y en el cursor
* L : fijar el límite superior de Y al valor Y en el cursor
* g : activar/desactivar la cuadrícula
* f : activar/desactivar la leyenda

Panorámica
----------
* flecha izquierda : desplazar a la izquierda
* flecha derecha : desplazar a la derecha
* flecha arriba : desplazar hacia arriba
* flecha abajo : desplazar hacia abajo
