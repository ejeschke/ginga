El modo Contraste activa asignaciones de teclas que permiten ajustar el
contraste de una imagen en un visor de imágenes de Ginga.

Entrar en el modo mediante
--------------------------
* Espacio, luego «t»

Salir del modo mediante
-----------------------
* Esc

Asignaciones predeterminadas del modo
-------------------------------------
* T : restaurar el contraste a los valores predeterminados
* arrastrar con el izquierdo : ajustar el contraste
  * Desplazar/estirar el mapa de colores de forma interactiva (contraste y sesgo).
  * Mover izquierda/derecha controla el desplazamiento, arriba/abajo el estiramiento.
  * Suelte el botón cuando el contraste sea satisfactorio.
* clic derecho : restaurar el contraste a los valores predeterminados
* rueda de desplazamiento : cambiar el contraste (añada Ctrl para un ajuste más fino)
* Mayús + rueda de desplazamiento : cambiar el brillo (añada Ctrl para más finura)
* Ctrl + gesto de panorámica : cambiar el contraste
* Mayús + gesto de panorámica : cambiar el brillo
