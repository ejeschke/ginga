El modo Cuts activa asignaciones de teclas que permiten ajustar los niveles
de corte inferior y superior de una imagen en un visor de imágenes de Ginga.

Entrar en el modo mediante
--------------------------
* Espacio, luego «s»

Salir del modo mediante
-----------------------
* Esc

Asignaciones predeterminadas del modo
-------------------------------------
* l : fijar el nivel de corte inferior al valor del píxel bajo el cursor
* k : fijar el nivel de corte superior al valor del píxel bajo el cursor
* S : fijar los niveles de corte inferior y superior a los valores mín/máx de
  la imagen
* A : fijar los niveles de corte inferior y superior a 0, 255;
  útil sobre todo para imágenes RGB estándar
* a : realizar un corte automático con el algoritmo y los parámetros de corte
  automático seleccionados actualmente
* b, flecha arriba : seleccionar el algoritmo de corte automático anterior de la lista
* n, flecha abajo : seleccionar el siguiente algoritmo de corte automático de la lista
* dos puntos : activar o desactivar el corte automático para nuevas imágenes en este visor
* punto y coma : fijar el corte automático para nuevas imágenes en «override» en este visor
* rueda de desplazamiento : ajustar el contraste comprimiendo o estirando los niveles;
  una dirección comprime, la otra estira
* Ctrl + desplazar : ajustar el microcontraste comprimiendo o estirando los niveles;
  similar a desplazar, pero con menor cantidad de estiramiento/compresión
* Mayús + desplazar : cambiar el algoritmo de corte automático actual
* arrastrar con el izquierdo : ajustar los niveles moviendo el cursor;
  izquierda/derecha ajusta el nivel superior, arriba/abajo el inferior
* Mayús + arrastrar con el izquierdo : ajustar el nivel inferior moviendo el cursor;
  izquierda/derecha ajusta el nivel inferior
* Ctrl + arrastrar con el izquierdo : ajustar el nivel superior moviendo el cursor;
  izquierda/derecha ajusta el nivel superior
* clic derecho : realizar niveles automáticos (igual que «a»)
* gesto de pellizco : ampliar o reducir la separación entre los niveles de corte
  inferior y superior (similar a la rueda de desplazamiento)
* Ctrl + gesto de panorámica : subir o bajar el nivel de corte superior
* Mayús + gesto de panorámica : subir o bajar el nivel de corte inferior
