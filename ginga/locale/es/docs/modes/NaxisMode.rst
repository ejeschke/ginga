El modo Naxis activa asignaciones de teclas que permiten moverse por las
rebanadas de una pila de imágenes en un visor de imágenes de Ginga.

Entrar en el modo mediante
--------------------------
* Espacio, luego «n»

Salir del modo mediante
-----------------------
* Esc

Asignaciones predeterminadas del modo
-------------------------------------
* desplazar : seleccionar la rebanada anterior o siguiente del eje actual
* Ctrl + desplazar : seleccionar el eje anterior o siguiente como eje actual
* arrastrar con el izquierdo : seleccionar la rebanada en función del porcentaje
  del ancho del cursor/ventana
* flecha arriba/abajo : seleccionar el eje anterior o siguiente como eje actual
