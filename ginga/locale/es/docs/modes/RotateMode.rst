El modo Rotación activa asignaciones de teclas que permiten voltear o
intercambiar los ejes de una imagen, o rotarla, en un visor de imágenes de
Ginga.

Entrar en el modo mediante
--------------------------
* Espacio, luego «r»

Salir del modo mediante
-----------------------
* Esc

Asignaciones predeterminadas del modo
-------------------------------------
* «[» : (alternar) voltear la imagen en el eje X
* «{» : restaurar el eje X
* «]» : (alternar) voltear la imagen en el eje Y
* «}» : restaurar el eje Y
* barra invertida : (alternar) intercambiar los ejes X e Y
* «|» : restaurar los ejes intercambiados a la normalidad
* R : restablecer la rotación a 0 grados (no restablece volteos ni intercambios)
* t : restablecer cualquier volteo o intercambio
* punto : rotar la imagen incrementalmente +90 grados
* coma : rotar la imagen incrementalmente -90 grados
* o : orientar la imagen con el Norte arriba y el Este a la izquierda
* O : orientar la imagen con el Norte arriba y el Este a la derecha
* arrastrar con el izquierdo : rotar la imagen de forma interactiva
* clic derecho : restablecer la rotación a 0 grados (igual que R)
* gesto de rotación : rotar la imagen de forma interactiva
