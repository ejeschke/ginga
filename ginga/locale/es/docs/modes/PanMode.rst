El modo Pan activa asignaciones de teclas que permiten fijar la posición de
panorámica (la posición bajo el píxel central) y el nivel de zoom (escala) en
un visor de imágenes de Ginga.

Entrar en el modo mediante
--------------------------
* Espacio, luego «q»

Salir del modo mediante
-----------------------
* Esc

Asignaciones predeterminadas del modo
-------------------------------------
* más, igual : acercar un nivel de zoom
* menos, guion bajo : alejar un nivel de zoom
* 1-9,0 : hacer zoom al nivel N (0 es 10)
* Mayús + 1-9,0 : hacer zoom al nivel -N (0 es -10)
* acento grave : ajustar la imagen al tamaño de la ventana
* comillas dobles : activar/desactivar el ajuste automático para nuevas imágenes
* comilla simple : fijar el ajuste automático para nuevas imágenes en «override»
* p : desplazar a la posición bajo el cursor
* c : desplazar al centro de la imagen
* z : guardar el nivel de zoom (escala)
* r : desplazar al cursor y hacer zoom al nivel de escala guardado
* flecha izquierda/derecha/arriba/abajo : desplazar un pequeño porcentaje
* Mayús + flechas : desplazar un porcentaje muy pequeño
* Re Pág (Av Pág) : desplazar hacia arriba (abajo) un gran porcentaje de la pantalla
* Inicio (Fin) : desplazar hacia el borde superior (inferior) de la imagen
* «?» : activar/desactivar el centrado automático para nuevas imágenes en este visor
* «/» : fijar el centrado automático para nuevas imágenes en «override» en este visor
* desplazar : hacer zoom (escalar) la imagen
* arrastrar con el izquierdo : desplazar la vista
* arrastrar con el derecho : desplazar la vista con la cámara
* gesto de panorámica : desplazar la vista de forma proporcional al gesto
* gesto de pellizco : hacer zoom en la imagen de forma proporcional al gesto
