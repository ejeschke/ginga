El modo Dist activa asignaciones de teclas que permiten ajustar la
distribución de color de una imagen en un visor de imágenes de Ginga.

Estos algoritmos son similares a los perfiles de tipo «curvas»: «linear»,
«log», «power», «sqrt», «squared», «asinh», «sinh», «histeq».

Entrar en el modo mediante
--------------------------
* Espacio, luego «d»

Salir del modo mediante
-----------------------
* Esc

Asignaciones predeterminadas del modo
-------------------------------------
* D : restablecer el algoritmo de distribución de color a «linear»
* b, flecha arriba : seleccionar el algoritmo de distribución anterior de la lista
* n, flecha abajo : seleccionar el siguiente algoritmo de distribución de la lista
* rueda de desplazamiento : seleccionar el algoritmo de distribución de color al desplazar
* gesto de panorámica : seleccionar el algoritmo de distribución de color al deslizar
  (sugerencia: confirme la selección del algoritmo con las flechas arriba/abajo)
