El modo CMap activa asignaciones de teclas que permiten ajustar los mapas de
color e intensidad de una imagen en un visor de imágenes de Ginga.

Entrar en el modo mediante
--------------------------
* Espacio, luego «y»

Salir del modo mediante
-----------------------
* Esc

Asignaciones predeterminadas del modo
-------------------------------------
* Y : restablecer el mapa de colores al predeterminado (gris)
* r : restaurar el mapa de colores (deshacer inversión, rotación o estiramiento)
* I : invertir el mapa de colores actual (conservando rotación o estiramiento)
* flecha arriba : mapa de colores anterior de la lista (conserva rotación/estiramiento)
* flecha abajo : siguiente mapa de colores de la lista (conserva rotación/estiramiento)
* c : superponer un mapa de colores visible sobre la imagen
* i : restaurar el mapa de intensidad predeterminado «ramp»
* j, flecha izquierda : mapa de intensidad anterior de la lista
* k, flecha derecha : siguiente mapa de intensidad de la lista
* desplazar : elegir el mapa de colores de la lista
* Ctrl + desplazar : elegir el mapa de intensidad de la lista
* gesto de panorámica : elegir el mapa de colores de la lista (sugerencia: use
  las flechas arriba/abajo para confirmar la selección)
* arrastrar con el izquierdo : rotar el mapa de colores actual
* clic derecho : restaurar el mapa de colores (igual que «r»)
