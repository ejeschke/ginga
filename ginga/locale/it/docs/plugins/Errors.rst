Il plugin ``Errors`` segnala i messaggi di errore nel visualizzatore.

**Tipo di plugin: Globale**

``Errors`` è un plugin globale.  Può essere aperta una sola istanza.

**Uso**

Quando si verifica un errore in Ginga, il suo messaggio può essere segnalato
qui.

Questo plugin di solito non è configurato come chiudibile, ma l'utente può
renderlo tale impostando l'opzione « closeable » su True nel file di
configurazione; in tal caso i pulsanti Chiudi e Aiuto verranno aggiunti in
fondo all'interfaccia.
