Un plugin per generare una sovrapposizione di assi WCS nell'immagine caricata.

**Tipo di plugin: Locale**

``WCSAxes`` è un plugin locale, il che significa che è associato a un canale.
È possibile aprire un'istanza per ciascun canale.

**Uso**

Finché l'immagine possiede un WCS valido, verranno visualizzati gli assi WCS.
Usa l'interfaccia del plugin o il file di configurazione per personalizzare la
visualizzazione degli assi.
