La modalità Disegno attiva le assegnazioni che facilitano il disegno.

Entra nella modalità con
------------------------
* Spazio, poi « b »

Esci dalla modalità con
-----------------------
* Esc

Assegnazioni predefinite della modalità
---------------------------------------
* v : aggiungere un vertice a un poligono, mentre si disegna o si modifica una tela
* z : eliminare un vertice da un poligono, mentre si disegna o si modifica una tela
* x : eliminare un oggetto selezionato mentre si modifica una tela
* trascina con il sinistro : disegnare sulla tela con la forma definita
* trascina con il destro : disegnare sulla tela con la forma definita
