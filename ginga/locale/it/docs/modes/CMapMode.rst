La modalità CMap attiva le assegnazioni che permettono di regolare le mappe
di colore e di intensità di un'immagine in un visualizzatore Ginga.

Entra nella modalità con
------------------------
* Spazio, poi « y »

Esci dalla modalità con
-----------------------
* Esc

Assegnazioni predefinite della modalità
---------------------------------------
* Y : ripristinare la mappa colori predefinita (grigio)
* r : ripristinare la mappa colori (annullare inversione, rotazione o stiramento)
* I : invertire la mappa colori attuale (conservando rotazione o stiramento)
* freccia su : mappa colori precedente dell'elenco (conserva rotazione/stiramento)
* freccia giù : mappa colori successiva dell'elenco (conserva rotazione/stiramento)
* c : sovrapporre una mappa colori visibile sull'immagine
* i : ripristinare la mappa di intensità predefinita « ramp »
* j, freccia sinistra : mappa di intensità precedente dell'elenco
* k, freccia destra : mappa di intensità successiva dell'elenco
* scorri : scegliere la mappa colori dall'elenco
* Ctrl + scorri : scegliere la mappa di intensità dall'elenco
* gesto di panoramica : scegliere la mappa colori dall'elenco (suggerimento:
  usare le frecce su/giù per confermare la selezione)
* trascina con il sinistro : ruotare la mappa colori attuale
* clic destro : ripristinare la mappa colori (come « r »)
