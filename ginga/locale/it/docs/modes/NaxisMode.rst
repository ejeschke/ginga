La modalità Naxis attiva le assegnazioni che permettono di spostarsi tra le
sezioni di una pila di immagini in un visualizzatore Ginga.

Entra nella modalità con
------------------------
* Spazio, poi « n »

Esci dalla modalità con
-----------------------
* Esc

Assegnazioni predefinite della modalità
---------------------------------------
* scorri : selezionare la sezione precedente o successiva dell'asse attuale
* Ctrl + scorri : selezionare l'asse precedente o successivo come asse attuale
* trascina con il sinistro : selezionare la sezione in funzione della percentuale
  della larghezza cursore/finestra
* freccia su/giù : selezionare l'asse precedente o successivo come asse attuale
