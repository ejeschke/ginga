Capturer des images PNG ou JPEG de l'image du visualiseur de canal.

**Utilisation**

1. Sélectionnez le type de graphiques RGB pour la capture dans la liste
   déroulante « Type ».
2. Appuyez sur « Capturer » lorsque l'image du canal est telle que vous voulez
   la capturer.

Une copie de l'image RGB sera chargée dans le visualiseur ``ScreenShot``.
Vous pouvez effectuer un panoramique et un zoom dans le visualiseur
``ScreenShot`` comme dans un visualiseur Ginga normal pour examiner les détails
(p. ex. voir la différence agrandie entre les formats JPEG et PNG).

3. Répétez (1) et (2) jusqu'à obtenir l'image souhaitée.
4. Saisissez un chemin valide pour un nouveau fichier dans la zone de texte
   « Dossier ».
5. Saisissez un nom valide pour un nouveau fichier dans la zone de texte
   « Nom ».  Il n'est pas nécessaire d'ajouter l'extension du fichier ; elle
   sera ajoutée si nécessaire.
6. Appuyez sur le bouton « Enregistrer ».  Le fichier sera enregistré à
   l'emplacement que vous avez indiqué.

**Remarques**

* Le PNG offre moins d'artefacts pour les graphiques superposés, mais les
  fichiers sont plus volumineux que le JPEG.
* Le bouton « Centrer » centrera l'image de capture ; « Ajuster » réglera le
  zoom pour l'ajuster à la fenêtre ; et « Effacer » effacera l'image.  Appuyez
  sur « Complet » pour zoomer à 100 % des pixels (échelle 1:1).
* La case « Taille de l'écran » (cochée par défaut) enregistrera l'image
  exactement à la taille de la fenêtre du visualiseur de canal.  Pour enregistrer
  à une taille différente, décochez cette case et définissez la taille via les
  cases « Largeur » et « Hauteur ».
* La fonction « Verrouiller le rapport d'aspect » ne fonctionne que si « Taille
  de l'écran » est décochée ; si elle est activée, modifier la largeur ou la
  hauteur modifie l'autre paramètre afin de conserver le rapport affiché dans la
  case « Rapport ».
