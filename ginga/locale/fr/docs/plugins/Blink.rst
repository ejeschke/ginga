``Blink`` fait défiler les images affichées dans un canal à une vitesse choisie
par l'utilisateur.  Il peut aussi basculer entre les canaux de l'espace de
travail principal.  Dans les deux cas, le but principal est de comparer et
d'opposer visuellement les images (au sein d'un canal ou entre canaux) sur une
courte durée -- comme cligner des yeux.

**Type de plugin : Local ou Global**

``Blink`` peut être invoqué soit comme plugin local, auquel cas il fait défiler
les images du canal, soit comme plugin global, auquel cas il fait défiler les
canaux.

Les plugins locaux sont lancés depuis le bouton « Opération », tandis que les
plugins globaux sont lancés depuis le menu « Plugins ».

**Utilisation**

Réglez l'intervalle entre les changements d'image en secondes dans la case
« Intervalle ».  Appuyez ensuite sur « Démarrer le clignotement » pour lancer
le défilement chronométré, et « Arrêter le clignotement » pour l'arrêter.

Vous pouvez modifier le nombre dans « Intervalle » et appuyer sur ``Enter`` pour
changer dynamiquement la durée du cycle pendant qu'il tourne.
