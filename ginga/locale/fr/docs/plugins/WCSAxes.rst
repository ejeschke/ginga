Un plugin pour générer une superposition d'axes WCS dans l'image chargée.

**Type de plugin : Local**

``WCSAxes`` est un plugin local, ce qui signifie qu'il est associé à un canal.
Une instance peut être ouverte pour chaque canal.

**Utilisation**

Tant que l'image possède un WCS valide, les axes WCS sont affichés.
Utilisez l'interface du plugin ou le fichier de configuration pour personnaliser
l'affichage des axes.
