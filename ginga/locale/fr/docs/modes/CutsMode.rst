Le mode Cuts active des raccourcis permettant d'ajuster les niveaux de coupure
inférieur et supérieur d'une image dans un visualiseur d'images Ginga.

Entrer dans le mode par
-----------------------
* Espace, puis « s »

Quitter le mode par
-------------------
* Échap

Raccourcis par défaut du mode
-----------------------------
* l : fixer le niveau de coupure bas à la valeur du pixel sous le curseur
* k : fixer le niveau de coupure haut à la valeur du pixel sous le curseur
* S : fixer les niveaux de coupure bas et haut aux valeurs min/max de l'image
* A : fixer les niveaux de coupure bas et haut à 0, 255 ;
  utile surtout pour les images RGB standard
* a : effectuer une coupure automatique avec l'algorithme et les paramètres
  de coupure automatique actuellement sélectionnés
* b, flèche haut : sélectionner l'algorithme de coupure automatique précédent
* n, flèche bas : sélectionner l'algorithme de coupure automatique suivant
* deux-points : activer/désactiver la coupure automatique pour les nouvelles images
* point-virgule : régler la coupure automatique des nouvelles images sur « override »
* molette : ajuster le contraste en resserrant ou en étirant les niveaux ;
  une direction resserre, l'autre étire
* Ctrl + défiler : ajuster le micro-contraste en resserrant ou en étirant ;
  comme la molette, mais l'amplitude d'étirement/resserrement est réduite
* Maj + défiler : changer l'algorithme de coupure automatique actuel
* glisser gauche : ajuster les niveaux en déplaçant le curseur ;
  gauche/droite ajuste le niveau haut, haut/bas ajuste le niveau bas
* Maj + glisser gauche : ajuster le niveau bas en déplaçant le curseur ;
  gauche/droite ajuste le niveau bas
* Ctrl + glisser gauche : ajuster le niveau haut en déplaçant le curseur ;
  gauche/droite ajuste le niveau haut
* clic droit : effectuer des niveaux automatiques (comme « a »)
* geste de pincement : élargir ou réduire l'écart entre les niveaux de coupure
  bas et haut (comme la molette)
* Ctrl + geste de panoramique : monter ou baisser le niveau de coupure haut
* Maj + geste de panoramique : monter ou baisser le niveau de coupure bas
