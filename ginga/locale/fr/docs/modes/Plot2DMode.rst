Le mode Plot2D active des raccourcis permettant de fixer la position de
panoramique et le niveau de zoom (échelle) dans un visualiseur de graphiques
Matplotlib (module ginga.plot.PlotView).

Entrer dans le mode par
-----------------------
* Espace, puis « p »

Quitter le mode par
-------------------
* Échap

Raccourcis par défaut du mode
-----------------------------
* Maj + clic gauche : fixer la position de panoramique
* clic du milieu : fixer la position de panoramique
* défiler : zoomer/dézoomer
* Ctrl + défiler : zoomer/dézoomer uniquement l'axe X
* Maj + défiler : zoomer/dézoomer uniquement l'axe Y (sur MacOS, pavé tactile uniquement)
* Alt + défiler (souris) : zoomer/dézoomer uniquement l'axe Y (touche Option sur Mac)
* Alt + défiler : zoomer/dézoomer au niveau du curseur

Raccourcis clavier
------------------

Zoom
----
* égal : zoomer d'un niveau
* Ctrl + égal : zoomer l'axe X d'un niveau
* plus (Maj + égal) : zoomer l'axe Y d'un niveau
* moins : dézoomer d'un niveau
* Ctrl + moins : dézoomer l'axe X d'un niveau
* tiret bas (Maj + moins) : dézoomer l'axe Y d'un niveau
* 9 : dézoomer en conservant la position du curseur
* Ctrl + 9 : dézoomer l'axe X en conservant la position du curseur
* parenthèse gauche (Maj + 9) : dézoomer l'axe Y en conservant la position du curseur
* 0 : zoomer en conservant la position du curseur
* Ctrl + 0 : zoomer l'axe X en conservant la position du curseur
* parenthèse droite (Maj + 0) : zoomer l'axe Y en conservant la position du curseur
* accent grave : ajuster les axes X et Y à la fenêtre
* 1 : ajuster uniquement l'axe X à la fenêtre
* 2 : ajuster uniquement l'axe Y à la fenêtre
* k : fixer la borne inférieure de X à la valeur X au curseur
* l : fixer la borne supérieure de X à la valeur X au curseur
* K : fixer la borne inférieure de Y à la valeur Y au curseur
* L : fixer la borne supérieure de Y à la valeur Y au curseur
* g : activer/désactiver la grille
* f : activer/désactiver la légende

Panoramique
-----------
* flèche gauche : panoramique à gauche
* flèche droite : panoramique à droite
* flèche haut : panoramique vers le haut
* flèche bas : panoramique vers le bas
