Le mode Dessin active des raccourcis qui facilitent le dessin.

Entrer dans le mode par
-----------------------
* Espace, puis « b »

Quitter le mode par
-------------------
* Échap

Raccourcis par défaut du mode
-----------------------------
* v : ajouter un sommet à un polygone, pendant le dessin ou l'édition d'un canevas
* z : supprimer un sommet d'un polygone, pendant le dessin ou l'édition d'un canevas
* x : supprimer un objet sélectionné pendant l'édition d'un canevas
* glisser gauche : dessiner sur le canevas selon la forme définie
* glisser droit : dessiner sur le canevas selon la forme définie
