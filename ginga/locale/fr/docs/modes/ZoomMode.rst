Le mode Zoom active des raccourcis permettant de fixer la position de
panoramique (le pixel central) et le niveau de zoom (échelle) dans un
visualiseur d'images Ginga.

Il diffère du mode Pan par les types de contrôles de défilement et de
panoramique.

Entrer dans le mode par
-----------------------
* Espace, puis « w »

Quitter le mode par
-------------------
* Échap

Raccourcis par défaut du mode
-----------------------------
* glisser milieu : déplacer librement dans l'image
* clic gauche : fixer le panoramique sous le curseur et zoomer
* clic droit : dézoomer
* geste de panoramique : zoomer/dézoomer l'image
* Maj + geste de panoramique : zoomer/dézoomer l'image avec l'origine fixée
