Le mode CMap active des raccourcis permettant d'ajuster les cartes de couleur
et d'intensité d'une image dans un visualiseur d'images Ginga.

Entrer dans le mode par
-----------------------
* Espace, puis « y »

Quitter le mode par
-------------------
* Échap

Raccourcis par défaut du mode
-----------------------------
* Y : réinitialiser la palette de couleurs par défaut (gris)
* r : restaurer la palette (annuler inversion, rotation ou étirement)
* I : inverser la palette actuelle (en conservant rotation ou étirement)
* flèche haut : palette précédente de la liste (conserve rotation/étirement)
* flèche bas : palette suivante de la liste (conserve rotation/étirement)
* c : superposer une palette de couleurs visible sur l'image
* i : restaurer la carte d'intensité par défaut « ramp »
* j, flèche gauche : carte d'intensité précédente de la liste
* k, flèche droite : carte d'intensité suivante de la liste
* défiler : choisir la palette de couleurs dans la liste
* Ctrl + défiler : choisir la carte d'intensité dans la liste
* geste de panoramique : choisir la palette dans la liste (astuce : utilisez
  les flèches haut/bas pour valider la sélection)
* glisser gauche : faire pivoter la palette actuelle
* clic droit : restaurer la palette (comme « r »)
