Le mode Naxis active des raccourcis permettant de parcourir les tranches d'une
pile d'images dans un visualiseur d'images Ginga.

Entrer dans le mode par
-----------------------
* Espace, puis « n »

Quitter le mode par
-------------------
* Échap

Raccourcis par défaut du mode
-----------------------------
* défiler : sélectionner la tranche précédente ou suivante de l'axe actuel
* Ctrl + défiler : sélectionner l'axe précédent ou suivant comme axe actuel
* glisser gauche : sélectionner la tranche selon le pourcentage de la largeur
  curseur/fenêtre
* flèche haut/bas : sélectionner l'axe précédent ou suivant comme axe actuel
