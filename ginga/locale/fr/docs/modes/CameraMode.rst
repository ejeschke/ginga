Le mode Caméra active des raccourcis permettant de manipuler la caméra dans un
visualiseur d'images Ginga fonctionnant avec un backend OpenGL.

.. note:: Le mode Caméra ne fonctionne que si le visualiseur utilise un
          backend OpenGL.

Entrer dans le mode par
-----------------------
* Espace, puis « c »

Quitter le mode par
-------------------
* Échap

Raccourcis par défaut du mode
-----------------------------
* s : enregistrer les réglages actuels de la caméra pour les restaurer ensuite
* r : rétablir les réglages de caméra enregistrés
* 3 : passer la vue de la caméra en mode 3D
* défiler : suivi de la caméra (avancer et reculer)
* glisser gauche : faire orbiter la vue avec la caméra
* glisser droit : déplacer la vue avec la caméra
