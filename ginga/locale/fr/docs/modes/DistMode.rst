Le mode Dist active des raccourcis permettant d'ajuster la distribution de
couleur d'une image dans un visualiseur d'images Ginga.

Ces algorithmes ressemblent à des profils de type « courbes » : « linear »,
« log », « power », « sqrt », « squared », « asinh », « sinh », « histeq ».

Entrer dans le mode par
-----------------------
* Espace, puis « d »

Quitter le mode par
-------------------
* Échap

Raccourcis par défaut du mode
-----------------------------
* D : réinitialiser l'algorithme de distribution de couleur sur « linear »
* b, flèche haut : sélectionner l'algorithme de distribution précédent
* n, flèche bas : sélectionner l'algorithme de distribution suivant
* molette : choisir l'algorithme de distribution de couleur en faisant défiler
* geste de panoramique : choisir l'algorithme en balayant (astuce : validez la
  sélection de l'algorithme avec les flèches haut/bas)
