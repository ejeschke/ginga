Le mode Rotation active des raccourcis permettant de retourner ou d'échanger
les axes d'une image, ou de la faire pivoter, dans un visualiseur d'images
Ginga.

Entrer dans le mode par
-----------------------
* Espace, puis « r »

Quitter le mode par
-------------------
* Échap

Raccourcis par défaut du mode
-----------------------------
* « [ » : (basculer) retourner l'image sur l'axe X
* « { » : restaurer l'axe X
* « ] » : (basculer) retourner l'image sur l'axe Y
* « } » : restaurer l'axe Y
* barre oblique inverse : (basculer) échanger les axes X et Y
* « | » : rétablir les axes échangés à la normale
* R : réinitialiser la rotation à 0 deg (ne réinitialise pas retournements/échanges)
* t : réinitialiser les retournements ou échanges
* point : faire pivoter l'image de +90 deg par incréments
* virgule : faire pivoter l'image de -90 deg par incréments
* o : orienter l'image avec le Nord en haut et l'Est à gauche
* O : orienter l'image avec le Nord en haut et l'Est à droite
* glisser gauche : faire pivoter l'image de manière interactive
* clic droit : réinitialiser la rotation à 0 deg (comme R)
* geste de rotation : faire pivoter l'image de manière interactive
