Le mode Contraste active des raccourcis permettant d'ajuster le contraste
d'une image dans un visualiseur d'images Ginga.

Entrer dans le mode par
-----------------------
* Espace, puis « t »

Quitter le mode par
-------------------
* Échap

Raccourcis par défaut du mode
-----------------------------
* T : rétablir le contraste par défaut
* glisser gauche : ajuster le contraste
  * Décaler/étirer la palette de manière interactive (contraste et biais).
  * Gauche/droite contrôle le décalage, haut/bas contrôle l'étirement.
  * Relâchez le bouton lorsque le contraste vous convient.
* clic droit : rétablir le contraste par défaut
* molette : modifier le contraste (ajoutez Ctrl pour un réglage plus fin)
* Maj + molette : modifier la luminosité (ajoutez Ctrl pour un réglage plus fin)
* Ctrl + geste de panoramique : modifier le contraste
* Maj + geste de panoramique : modifier la luminosité
