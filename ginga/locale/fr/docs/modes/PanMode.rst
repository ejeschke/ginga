Le mode Pan active des raccourcis permettant de fixer la position de
panoramique (la position sous le pixel central) et le niveau de zoom (échelle)
dans un visualiseur d'images Ginga.

Entrer dans le mode par
-----------------------
* Espace, puis « q »

Quitter le mode par
-------------------
* Échap

Raccourcis par défaut du mode
-----------------------------
* plus, égal : zoomer d'un niveau
* moins, tiret bas : dézoomer d'un niveau
* 1-9,0 : zoomer au niveau N (0 vaut 10)
* Maj + 1-9,0 : zoomer au niveau -N (0 vaut -10)
* accent grave : ajuster l'image à la taille de la fenêtre
* guillemet : activer/désactiver l'ajustement automatique des nouvelles images
* apostrophe : régler l'ajustement automatique des nouvelles images sur « override »
* p : effectuer un panoramique vers la position sous le curseur
* c : effectuer un panoramique vers le centre de l'image
* z : enregistrer le niveau de zoom (échelle)
* r : panoramique vers le curseur et zoom au niveau d'échelle enregistré
* flèche gauche/droite/haut/bas : panoramique d'un petit pourcentage
* Maj + flèches : panoramique d'un très petit pourcentage
* Page préc. (Page suiv.) : panoramique vers le haut (bas) d'un grand pourcentage
* Début (Fin) : panoramique vers le haut (bas) de l'image
* « ? » : activer/désactiver le centrage automatique des nouvelles images
* « / » : régler le centrage automatique des nouvelles images sur « override »
* défiler : zoomer (mettre à l'échelle) l'image
* glisser gauche : déplacer la vue
* glisser droit : déplacer la vue avec la caméra
* geste de panoramique : déplacer la vue proportionnellement au geste
* geste de pincement : zoomer l'image proportionnellement au geste
