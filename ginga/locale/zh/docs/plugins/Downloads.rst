
Ginga 参考查看器的下载 GUI。

**插件类型：全局**

``Download`` 是全局插件。只能打开一个实例。

**用法**

打开此插件以监控 URI 下载的进度。使用「Plugins」或「Operations」菜单启动它，并从
「Util」类别下选择「Downloads」插件。

如果您想发起下载，只需将 URI 拖到通道图像查看器或 ``Thumbs`` 窗格中。

您可以随时通过单击某个下载条目的「清除」按钮来移除有关该下载的信息。您可以通过
单击底部的「全部清除」按钮来清除所有下载的条目。

目前，无法取消正在进行的下载。

**设置**

``auto_clear_download`` 选项如果设置为 `True`，将使下载完成时下载条目自动从窗格中
删除。它不会删除任何已下载的文件。

下载文件夹可由用户通过在 ~/.ginga/general.cfg 中为「download_folder」设置分配一个
值来定义。如果未分配，则默认为平台特定的默认临时目录中的一个文件夹（如 Python
「tempfile」模块所述）。
