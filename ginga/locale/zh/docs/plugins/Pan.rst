``Pan`` 插件提供一幅小的平移图像，给出上次获得焦点的通道图像的整体「鸟瞰」视图。
如果通道图像放大到 2 倍或更大，平移区域会在 ``Pan`` 图像中以矩形图示。

**插件类型：本地**

``Pan`` 是全局插件。只能打开一个实例。

**用法**

通道图像可以通过单击和/或拖动来放置矩形以进行平移。使用鼠标右键拖出矩形会迫使
通道图像查看器尝试匹配该区域（会考虑所绘矩形与窗口尺寸之间宽高比的差异）。在
``Pan`` 图像中滚动会缩放通道图像。

``Pan`` 图像的颜色/强度映射表和剖切电平会在相应通道图像中更改时更新。
如果通道中正在查看的 FITS HDU 中存在有效的 WCS 元数据，``Pan`` 图像还会显示
世界坐标系（WCS）罗盘。

``Pan`` 插件通常作为子窗格出现在「Info」选项卡下，紧邻 ``Info`` 插件。

此插件通常未配置为可关闭，但用户可以通过在配置文件中将「closeable」设置设为
True 来使其可关闭；届时界面底部将添加“关闭”和“帮助”按钮。
