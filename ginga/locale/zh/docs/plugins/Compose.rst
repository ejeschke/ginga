
用于从组成的单色图像合成 RGB 图像的插件。

**插件类型：本地**

``Compose`` 是本地插件，这意味着它与某个通道相关联。
可以为每个通道打开一个实例。

**用法**

从「Operation->RGB」（下方）或「Plugins->RGB」（上方）菜单启动 ``Compose``
插件。该选项卡应显示在右侧查看器的「Dialogs」选项卡下，名为「IMAGE:Compose」。

1. 从「Compose Type」下拉列表中选择您要进行的合成类型：「RGB」用于将三幅单色图像
   合成为一幅彩色图像，「Alpha」用于将一系列图像作为图层合成，每个图层具有不同的
   alpha 值。
2. 按「新建图像」开始合成一幅新图像。

***对于 RGB 合成***

1. 将构成 R、G、B 平面的三幅组成图像拖到「Preview」窗口——按 R（红）、G（绿）、
   B（蓝）的顺序拖动它们。或者，您可以逐幅将图像加载到通道查看器中，并在每幅之后
   按「从通道插入」（同样，按 R、G、B 的顺序进行）。

在插件 GUI 中，R、G、B 图像应显示为插件「Layers」区域中的三个滑块控件，预览应显示
在滑块设置下合成图像外观的低分辨率版本。

.. figure:: figures/compose-rgb.png
   :width: 800px
   :align: center
   :alt: 合成 RGB 图像

   合成 RGB 图像。

2. 使用 ``Compose`` 插件中的滑块调整每个图层的 alpha 级别；当您调整滑块时，预览
   图像应更新。
3. 当您看到满意的效果时，可以使用「另存为」按钮将其保存到文件（使用「jpeg」或
   「png」作为文件扩展名），或使用「保存到通道」按钮将其插入通道。

***对于 Alpha 合成***

对于 Alpha 类型合成，图像只是按堆栈中显示的顺序组合，图层 0 为最底层，后续图层
堆叠在其上。每个图层的 alpha 级别可通过滑块以与上文所述相同的方式进行调整。

.. figure:: figures/compose-alpha.png
   :width: 800px
   :align: center
   :alt: 对图像进行 Alpha 合成

   对图像进行 Alpha 合成。

1. 将构成图层的 N 幅组成图像拖到「Preview」窗口，或逐幅将图像加载到通道查看器中
   并在每幅之后按「从通道插入」（第一幅图像将位于堆栈的底部——图层 0）。
2. 使用 ``Compose`` 插件中的滑块调整每个图层的 alpha 级别；当您调整滑块时，预览
   图像应更新。
3. 当您看到满意的效果时，可以使用「另存为」按钮将其保存到文件（使用「fits」作为
   文件扩展名），或使用「保存到通道」按钮将其插入通道。

***一般说明***

- 预览窗口只是一个 ginga 小部件，因此所有常用绑定都适用；您可以使用鼠标和按键
  绑定设置色彩映射、剪切级别等。
