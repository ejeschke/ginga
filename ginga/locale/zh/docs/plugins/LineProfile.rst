用于绘制沿平分立方体的直线上像素值的插件。

**插件类型：本地**

``LineProfile`` 是本地插件，这意味着它与某个通道相关联。
可以为每个通道打开一个实例。

**用法**

.. warning::

   对可以选择哪些轴没有任何限制。
   因此，绘图可能毫无意义。

``LineProfile`` 插件用于多维（即 3D 或更高维）图像。它沿选定的轴绘制当前光标
位置处像素的值；或者，如果选择了某个区域，则绘制每一帧中的平均值。这可用于创建
普通的谱线轮廓。会在当前显示帧的数据点处放置一个标记。

显示的 X 轴使用 FITS 头中的 ``CRVAL*``、``CDELT*``、``CRPIX*``、``CTYPE*`` 和
``CUNIT*`` 关键字构建。如果其中任何关键字不可用，则该轴改为回退到 ``NAXIS*``
值。

显示的 Y 轴使用 ``BTYPE`` 和 ``BUNIT`` 构建。如果它们不可用，则只是将像素值
标记为「Signal」。

使用此插件的方法：

1. 选择一个轴。
2. 用光标选取一个点或绘制一个区域。
3. 如适用，使用 ``MultiDim`` 更改各轴的步进值。
