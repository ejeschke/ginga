``ColorMapPicker`` 插件用于以图形方式浏览并为通道图像查看器选择颜色映射表。

**插件类型：全局或本地**

``ColorMapPicker`` 是全局/本地混合插件，这意味着它可以任一方式调用。若作为本地
插件调用，则它与某个通道相关联，并且可以为每个通道打开一个实例。它也可以作为
全局插件打开。

**用法**

该插件的操作非常简单：颜色映射表以颜色条和标签的形式显示在插件的主视图窗格中。
单击其中任意一条颜色条，即可设置关联通道（若作为本地插件调用）或当前活动通道
（若作为全局插件调用）的颜色映射表。

您可以垂直滚动或使用滚动条在颜色条样本之间移动。

.. note:: 插件首次启动时，会生成一幅与所有可用颜色映射表对应的颜色条和标签的
          RGB 位图图像。这可能需要几秒钟，具体取决于已安装颜色映射表的数量。

          颜色映射表以应用「ramp」强度映射表的方式显示。
