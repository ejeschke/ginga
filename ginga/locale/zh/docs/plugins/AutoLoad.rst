``AutoLoad`` 是一个简单的插件，用于监视文件夹中的新文件，并在它们出现时自动
将其加载到通道中。

**插件类型：本地**

``AutoLoad`` 是本地插件，这意味着它与某个通道相关联。
可以为每个通道打开一个实例。

.. note:: 您需要安装 Python「watchdog」软件包才能使用此插件。

**用法**

* 要设置要监视的文件夹，请在「监视文件夹」字段中输入文件夹（目录）的路径，然后
  按 ENTER 或单击「设置」。
* 如果您需要区分将添加到此文件夹的文件，可以在「正则表达式」框中输入 Python
  正则表达式，然后单击「设置」。只会考虑名称与该模式匹配的文件。请注意，正则
  表达式仅针对文件名，而不针对文件夹路径的任何部分。
* 如果您想暂停自动加载，可以勾选标有「暂停自动加载」的复选框；这将停止所有自动
  加载。请注意，如果您随后取消勾选该复选框，则在此期间到达的文件将不会被加载。

.. note:: 监视位于网络驱动器上的文件夹可能有效，也可能无效。
