用于绘制画布形状（叠加图形）的插件。

**插件类型：本地**

``Drawing`` 是本地插件，这意味着它与某个通道相关联。
它不是单例，这意味着可以为每个通道打开多个实例。

**用法**

此插件可用于在图像显示上绘制许多不同的形状。处于「绘制」模式时，从下拉菜单中
选择一个形状，（如有需要）调整该形状的参数，然后使用鼠标左键在图像上绘制。您
可以选择在像素空间或 WCS 空间中绘制。

要移动或编辑现有形状，请分别将插件设置为「编辑」或「移动」模式。

要将所绘形状保存为掩码图像，请单击「创建掩码」按钮，您会看到在 Ginga 中创建了
一幅新的掩码图像。然后，使用 ``SaveImage`` 插件将其另存为单扩展 FITS。请注意，
掩码将采用所显示图像的大小。因此，要为不同的图像尺寸创建掩码，您需要多次重复
这些步骤。

在画布上绘制的形状可以以 astropy-regions 格式（与 DS9 区域兼容）加载和/或保存。
要使用它，您需要已安装 astropy-regions 软件包。只需在画布上绘制对象，坐标为
「data」（像素）或「wcs」。请注意，并非所有 Ginga 画布对象都能转换为区域形状，
某些属性可能不会被保存、可能被忽略，或者在其他软件中尝试加载区域形状时可能引发
错误。
