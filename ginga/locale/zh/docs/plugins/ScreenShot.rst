将通道查看器图像捕获为 PNG 或 JPEG 图像。

**用法**

1. 从「类型」组合框中为快照选择 RGB 图形类型。
2. 当通道图像达到您想捕获的样子时，按「快照」。

RGB 图像的副本将加载到 ``ScreenShot`` 查看器中。
您可以像在普通 Ginga 查看器中一样在 ``ScreenShot`` 查看器内平移和缩放，以查看
细节（例如，查看 JPEG 和 PNG 格式之间放大的差异）。

3. 重复 (1) 和 (2)，直到得到您想要的图像。
4. 在「文件夹」文本框中输入新文件的有效路径。
5. 在「名称」文本框中输入新文件的有效名称。
   无需添加文件扩展名；如有需要将会添加。
6. 按「保存」按钮。文件将保存到您指定的位置。

**注意事项**

* PNG 对叠加图形产生的伪影较少，但文件比 JPEG 大。
* 「居中」按钮会将快照图像居中；「适应」会设置缩放以使其适应窗口；「清除」会
  清除图像。按「完整」缩放到 100% 像素（1:1 缩放）。
* 「屏幕大小」复选框（默认勾选）会以恰好等于通道查看器窗口的大小保存图像。要以
  不同大小保存，请取消勾选此框，并通过「宽度」和「高度」框设置大小。
* 「锁定宽高比」功能仅在「屏幕大小」未勾选时有效；若启用，则更改宽度或高度会改变
  另一个参数，以保持「宽高比」框中显示的宽高比。
