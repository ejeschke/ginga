
``Info`` 插件为聚焦的通道图像提供一个常用的有用元数据窗格。常见信息包括一些元数据
头值、坐标、图像的尺寸、最小值和最大值等。当光标在图像上移动时，X、Y、Value、RA 和
DEC 值会更新以反映光标下的值。

**插件类型：全局**

``Info`` 是全局插件。只能打开一个实例。

**用法**

在 ``Info`` 界面底部是颜色分布和剪切级别控件。剪切级别框上方的选择器让您从多种分布
算法中选择，这些算法将图像中的值映射到色彩映射。选项为「linear」、「log」、「power」、
「sqrt」、「squared」、「asinh」、「sinh」和「histeq」（直方图均衡化）。

在此下方，显示低和高剪切级别并可进行调整。按「Auto Levels」按钮将根据当前的自动
剪切级别算法和通道首选项中定义的参数重新计算剪切级别。

在「Auto Levels」按钮下方，显示当前活动通道的「Cut New」、「Zoom New」和「Center
New」设置的状态。这些指示添加到通道的新图像将如何受到自动剪切级别、适应窗口以及
平移到图像中心的影响。

「Follow New」复选框控制查看器是否会自动显示添加到通道的新图像。「Raise New」复选框
控制在添加新图像时是否提升图像查看器窗口。这两个控件可能很有用，例如，如果外部程序
正在向查看器添加图像，而您希望防止在检查某幅特定图像时被打断。

作为全局插件，``Info`` 通过显示新通道的元数据来响应焦点切换到新通道。它通常出现在
用户界面的「Synopsis」选项卡下。

此插件通常未配置为可关闭，但用户可以通过在配置文件中将「closeable」设置设为 True
来使其可关闭——然后界面底部将添加「关闭」和「帮助」按钮。
