``Zoom`` 插件显示以关联通道图像中光标位置下方为中心的裁剪区域的放大图像。当光标
在图像上移动时，缩放图像会更新，以便仔细检查像素，或与其他插件操作配合进行精确
控制。

**插件类型：全局**

``Zoom`` 是全局插件。只能打开一个实例。

**用法**

缩放窗口的放大倍数可通过调整「缩放量」滑块来更改。

有两种操作模式——绝对缩放和相对缩放：

* 在绝对模式下，缩放量精确控制裁剪区域中显示的缩放级别；例如，通道图像可能被放大
  到 10X，但如果缩放量设置为 3X，则缩放图像将仅显示 3X 图像。

* 在相对模式下，缩放量设置被解释为相对于通道图像的缩放设置。如果缩放量设置为 3X
  且通道图像缩放到 10X，则显示的缩放图像将为 13X（10X + 3X）。请注意，缩放量设置
  可以 < 1，因此在通道图像 3X 缩放时设置 1/3X 将产生 1X 缩放图像。

「刷新间隔」设置控制 ``Zoom`` 插件在更新缩放图像时对光标移动的响应速度。该值以
毫秒为单位指定。

.. tip:: 通常，设置较小的刷新间隔会*改善*缩放图像的整体响应性，默认值 20 是一个
         合理的值。如果缩放图像看起来太卡顿或与通道图像窗口中的鼠标移动不同步，
         您可以尝试调整该值。

「默认值」按钮可恢复控件的默认设置。

它可使用 ``~/.ginga/plugin_Zoom.cfg`` 进行自定义，其中 ``~`` 是您的 HOME 目录：

.. code-block:: Python

  #
  # Zoom plugin preferences file
  #
  # Place this in file under ~/.ginga with the name "plugin_Zoom.cfg"

  # default zoom level
  zoom_amount = 3

  # refresh interval (sec)
  # NOTE: usually a small delay speeds things up
  refresh_interval = 0.02
