``Header`` 插件提供与图像相关联的元数据的列表。

**插件类型：全局**

``Header`` 是全局插件。只能打开一个实例。

**用法**

``Header`` 插件显示图像的 FITS 关键字元数据。最初仅显示主 HDU 的元数据。不过，
与 ``MultiDim`` 插件配合使用时，将显示其他 HDU 的元数据。详见 ``MultiDim``。

如果勾选了界面左下角的「可排序」复选框，则单击列标题会按该列的值对表格排序，
这对于快速定位某个特定关键字可能很有用。

「包含主头」复选框用于切换是否包含主 HDU 的关键字。如果图像是使用不保存主头的
选项创建的，则此选项可能被禁用。
