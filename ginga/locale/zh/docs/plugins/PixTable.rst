
``PixTable`` 提供了一种检查或监控某个区域内像素值的方法。

**插件类型：本地**

``PixTable`` 是本地插件，这意味着它与某个通道相关联。
可以为每个通道打开一个实例。

**基本用法**

在最基本的用法中，只需在通道查看器中移动光标；一个像素值数组将出现在插件界面的
「Pixel Values」显示中。中心值被高亮显示，它对应于光标下的值。

您可以从最左侧的组合框控件中选择 3x3、5x5、7x7 或 9x9 网格。调整「字体大小」控件
可能有助于防止数组值在两侧被截断。您也可以放大插件工作区以查看更多表格内容。

.. note:: 如果图像被翻转、转置或旋转，所显示值表的顺序不一定与通道查看器一致。

**使用标记**

当您设置并选择一个标记时，像素值将显示在标记周围而不是光标周围。可以有任意数量的
标记，每个标记都用一个带编号的「X」标注。只需更改标记下拉控件即可选择不同的标记
并查看其周围的值。当前选中的标记以不同于其他标记的颜色显示。

即使加载了新图像，标记也会保持在原位置，并显示新图像的值。这样，如果图像频繁更新，
您可以监控某个点周围的区域。

如果选中「平移到标记」复选框，那么当您在标记控件中选择不同的标记时，通道查看器将
平移到该标记。这对于在多幅不同图像中检查相同的点很有用，尤其是在紧密放大图像时。

.. note:: 如果您将标记控件改回「None」，则像素表将随着您在查看器中移动光标而再次
          更新。

「Caption」框可用于设置一个文本注释，在创建下一个标记时它将附加到标记标签上。例如，
这可用于标注图像中的某个特征。

**删除标记**

要删除标记，请在标记控件中选择它，然后按标有「删除」的按钮。要删除所有标记，请按
标有「全部删除」的按钮。

**移动标记**

当「移动」单选按钮被选中且有一个标记被选中时，在图像中任意位置单击或拖动将把标记
移动到该位置并更新像素表。如果当前没有选中任何标记，则会创建一个新标记并移动它。

**绘制标记**

当「绘制」单选按钮被选中时，单击并拖动会创建一个新标记。拖动越长，「X」的半径越大。

**编辑标记**

在选择了某个标记后，当「编辑」单选按钮被选中时，您可以拖动标记的控制点以增大 X
臂的半径，或者拖动边界框以移动标记。如果未显示编辑控制点，只需单击标记的中心即可
启用它们。

**特殊键**

在「移动」模式下，以下键处于活动状态：
- 「n」将在光标所在处放置一个新标记
- 「m」将把当前标记（如有）移动到光标所在处
- 「d」将删除当前标记（如有）
- 「j」将选择上一个标记（如有）
- 「k」将选择下一个标记（如有）

**用户配置**

它可使用 ``~/.ginga/plugin_PixTable.cfg`` 进行自定义，其中 ``~`` 是您的 HOME
目录：

.. code-block:: Python

  #
  # PixTable plugin preferences file
  #
  # Place this in file under ~/.ginga with the name "plugin_PixTable.cfg"

  # Default font
  font = 'fixed'

  # Default font size
  fontsize = 12

  # default size for mark point radius
  mark_radius = 10

  # style of point to draw
  mark_style = 'cross'

  # color of non-selected marks
  mark_color = 'purple'

  # color of selected mark
  select_color = 'cyan'

  # whether to update the pixel table when moving a mark around
  drag_update = True
