此插件为参考查看器提供命令行界面。

.. note:: 该命令行用于插件 UI *内部*。如果您要查找*远程*命令行界面，请参见
          ``RC`` 插件。

**插件类型：全局**

``Command`` 是全局插件。只能打开一个实例。

**用法**

获取命令和参数列表::

        g> help

执行 shell 命令::

        g> !cmd arg arg ...

**注意事项**

一个特别强大的工具是使用 ``reload_local`` 和 ``reload_global`` 命令，在开发某个
插件时重新加载该插件。这样就无需重启参考查看器并费力地重新加载数据等。只需关闭
插件，执行相应的「reload」命令（参见帮助！），然后再次启动该插件。

.. note:: 如果您修改了插件本身*以外*的模块，这些命令不会重新加载它们。
