``WCSMatch`` 是 Ginga 图像查看器的全局插件，可让您出于查看目的，使用图像的世界
坐标系（WCS）粗略对齐具有不同比例和方向的图像。

**插件类型：全局**

``WCSMatch`` 是全局插件。只能打开一个实例。

**用法**

要使用它，只需启动插件，然后从插件界面中标有「参考通道」的下拉菜单中选择一个
通道。该通道中包含的图像将用作同步其他通道图像的参考。

各通道将在查看方面（平移、比例（缩放）、变换（翻转）和旋转）进行同步。可勾选或
不勾选「匹配平移」、「匹配比例」、「匹配变换」和「匹配旋转」复选框，以控制在
通道之间同步哪些属性。

要完全「解锁」同步，只需从「参考通道」下拉菜单中选择「None」。

目前无法限制受该插件影响的通道。
