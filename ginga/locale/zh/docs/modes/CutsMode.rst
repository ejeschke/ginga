Cuts 模式启用可在 Ginga 图像查看器中调整图像下限和上限剖切电平的按键分配。

进入模式的方式
--------------
* 空格，然后「s」

退出模式的方式
--------------
* Esc

模式的默认按键分配
------------------
* l : 将下限剖切电平设为光标下的像素值
* k : 将上限剖切电平设为光标下的像素值
* S : 将下限和上限剖切电平设为图像的最小/最大值
* A : 将下限和上限剖切电平设为 0、255；主要对标准 RGB 图像有用
* a : 使用当前选定的自动剖切算法和参数执行自动剖切
* b、上箭头 : 选择列表中的上一个自动剖切算法
* n、下箭头 : 选择列表中的下一个自动剖切算法
* 冒号 : 在此查看器中切换新图像自动剖切的开/关
* 分号 : 在此查看器中将新图像的自动剖切设为「override」
* 滚轮 : 通过压缩或拉伸电平来调整对比度；一个方向压缩，另一个方向拉伸
* Ctrl + 滚动 : 通过压缩或拉伸电平来微调对比度；与滚动类似，但拉伸/压缩幅度更小
* Shift + 滚动 : 更改当前自动剖切算法
* 左键拖动 : 移动光标调整电平；左右调整上限，上下调整下限
* Shift + 左键拖动 : 移动光标调整下限；左右调整下限
* Ctrl + 左键拖动 : 移动光标调整上限；左右调整上限
* 右键单击 : 执行自动电平（同「a」）
* 捏合手势 : 加宽或收窄下限与上限剖切电平之间的间隔（类似滚轮）
* Ctrl + 平移手势 : 上下更改上限剖切电平
* Shift + 平移手势 : 上下更改下限剖切电平
