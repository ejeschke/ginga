CMap 模式启用可在 Ginga 图像查看器中调整图像颜色映射表和强度映射表的
按键分配。

进入模式的方式
--------------
* 空格，然后「y」

退出模式的方式
--------------
* Esc

模式的默认按键分配
------------------
* Y : 将颜色映射表重置为默认（灰度）
* r : 恢复颜色映射表（撤销反转、旋转或拉伸）
* I : 反转当前颜色映射表（保留旋转或拉伸）
* 上箭头 : 设置列表中的上一个颜色映射表（保留旋转/拉伸）
* 下箭头 : 设置列表中的下一个颜色映射表（保留旋转/拉伸）
* c : 在图像上叠加可见的颜色映射表
* i : 恢复默认强度映射表「ramp」
* j、左箭头 : 设置列表中的上一个强度映射表
* k、右箭头 : 设置列表中的下一个强度映射表
* 滚动 : 从列表中选择颜色映射表
* Ctrl + 滚动 : 从列表中选择强度映射表
* 平移手势 : 从列表中选择颜色映射表（提示：用上/下箭头键确定选择）
* 左键拖动 : 旋转当前颜色映射表
* 右键单击 : 恢复颜色映射表（同「r」）
