Dist 模式启用可在 Ginga 图像查看器中调整图像颜色分布的按键分配。

这些算法类似于「曲线」类型的配置文件：「linear」、「log」、「power」、
「sqrt」、「squared」、「asinh」、「sinh」、「histeq」。

进入模式的方式
--------------
* 空格，然后「d」

退出模式的方式
--------------
* Esc

模式的默认按键分配
------------------
* D : 将颜色分布算法重置为「linear」
* b、上箭头 : 选择列表中的上一个分布算法
* n、下箭头 : 选择列表中的下一个分布算法
* 滚轮 : 通过滚动选择颜色分布算法
* 平移手势 : 通过滑动选择颜色分布算法（提示：用上/下箭头键确定）
