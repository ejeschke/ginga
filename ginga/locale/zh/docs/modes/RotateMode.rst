旋转模式启用可在 Ginga 图像查看器中翻转或交换图像的轴、或旋转图像的
按键分配。

进入模式的方式
--------------
* 空格，然后「r」

退出模式的方式
--------------
* Esc

模式的默认按键分配
------------------
* 「[」: （切换）沿 X 轴翻转图像
* 「{」: 恢复 X 轴
* 「]」: （切换）沿 Y 轴翻转图像
* 「}」: 恢复 Y 轴
* 反斜杠 : （切换）交换 X 轴和 Y 轴
* 「|」: 将交换的轴恢复正常
* R : 将旋转重置为 0 度（不重置翻转或交换）
* t : 重置任何翻转或交换
* 句点 : 将图像增量旋转 +90 度
* 逗号 : 将图像增量旋转 -90 度
* o : 使图像朝向为北在上、东在左
* O : 使图像朝向为北在上、东在右
* 左键拖动 : 交互式旋转图像
* 右键单击 : 将旋转重置为 0 度（同 R）
* 旋转手势 : 交互式旋转图像
