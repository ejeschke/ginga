Pan 模式启用可在 Ginga 图像查看器中设置平移位置（中央像素下的位置）和
缩放级别（比例）的按键分配。

进入模式的方式
--------------
* 空格，然后「q」

退出模式的方式
--------------
* Esc

模式的默认按键分配
------------------
* 加号、等号 : 放大一个缩放级别
* 减号、下划线 : 缩小一个缩放级别
* 1-9,0 : 缩放到级别 N（0 为 10）
* Shift + 1-9,0 : 缩放到级别 -N（0 为 -10）
* 反引号 : 将图像适应窗口大小
* 双引号 : 在此查看器中切换新图像自动适应的开/关
* 单引号 : 在此查看器中将新图像的自动适应设为「override」
* p : 平移到光标下的位置
* c : 平移到图像中心
* z : 保存缩放级别（比例）
* r : 平移到光标并缩放到已保存的比例级别
* 左/右/上/下箭头 : 平移一个较小的百分比
* Shift + 箭头 : 平移一个非常小的百分比
* PageUp（PageDown）: 向上（下）平移屏幕的一大部分
* Home（End）: 向图像的顶部（底部）方向平移
* 「?」: 在此查看器中切换新图像自动居中的开/关
* 「/」: 在此查看器中将新图像的自动居中设为「override」
* 滚动 : 缩放（缩放比例）图像
* 左键拖动 : 平移视图
* 右键拖动 : 用相机平移视图
* 平移手势 : 按手势比例平移视图
* 捏合手势 : 按手势比例缩放图像
