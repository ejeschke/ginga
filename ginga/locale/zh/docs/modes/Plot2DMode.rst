Plot2D 模式启用可在 Matplotlib 绘图查看器（ginga.plot.PlotView 模块）中
设置平移位置和缩放级别（比例）的按键分配。

进入模式的方式
--------------
* 空格，然后「p」

退出模式的方式
--------------
* Esc

模式的默认按键分配
------------------
* Shift + 左键单击 : 设置平移位置
* 中键单击 : 设置平移位置
* 滚动 : 放大/缩小
* Ctrl + 滚动 : 仅放大/缩小 X 轴
* Shift + 滚动 : 仅放大/缩小 Y 轴（在 MacOS 上仅触控板）
* Alt + 滚动（鼠标）: 仅放大/缩小 Y 轴（Mac 上为 Option 键）
* Alt + 滚动 : 在光标处放大/缩小

按键分配
--------

缩放
----
* 等号 : 放大一个缩放级别
* Ctrl + 等号 : 将 X 轴放大一个缩放级别
* 加号（Shift + 等号）: 将 Y 轴放大一个缩放级别
* 减号 : 缩小一个缩放级别
* Ctrl + 减号 : 将 X 轴缩小一个缩放级别
* 下划线（Shift + 减号）: 将 Y 轴缩小一个缩放级别
* 9 : 保持光标位置缩小
* Ctrl + 9 : 保持光标位置缩小 X 轴
* 左括号（Shift + 9）: 保持光标位置缩小 Y 轴
* 0 : 保持光标位置放大
* Ctrl + 0 : 保持光标位置放大 X 轴
* 右括号（Shift + 0）: 保持光标位置放大 Y 轴
* 反引号 : 将 X 轴和 Y 轴适应窗口
* 1 : 仅将 X 轴适应窗口
* 2 : 仅将 Y 轴适应窗口
* k : 将 X 下限范围设为光标处的 X 值
* l : 将 X 上限范围设为光标处的 X 值
* K : 将 Y 下限范围设为光标处的 Y 值
* L : 将 Y 上限范围设为光标处的 Y 值
* g : 切换网格开/关
* f : 切换图例开/关

平移
----
* 左箭头 : 向左平移
* 右箭头 : 向右平移
* 上箭头 : 向上平移
* 下箭头 : 向下平移
