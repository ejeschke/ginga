对比度模式启用可在 Ginga 图像查看器中调整图像对比度的按键分配。

进入模式的方式
--------------
* 空格，然后「t」

退出模式的方式
--------------
* Esc

模式的默认按键分配
------------------
* T : 将对比度恢复为默认值
* 左键拖动 : 调整对比度
  * 交互式平移/拉伸颜色映射表（对比度和偏置）。
  * 左右移动控制平移，上下控制拉伸。
  * 对比度满意时松开按键。
* 右键单击 : 将对比度恢复为默认值
* 滚轮 : 更改对比度（按住 Ctrl 可更精细地调整）
* Shift + 滚轮 : 更改亮度（按住 Ctrl 可更精细地调整）
* Ctrl + 平移手势 : 更改对比度
* Shift + 平移手势 : 更改亮度
