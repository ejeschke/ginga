Naxis 모드는 Ginga 이미지 뷰어에서 이미지 스택의 슬라이스 사이를 이동하는
키 할당을 활성화합니다.

모드 진입 방법
--------------
* 스페이스, 그다음 「n」

모드 종료 방법
--------------
* Esc

모드의 기본 키 할당
-------------------
* 스크롤 : 현재 축의 이전 또는 다음 슬라이스 선택
* Ctrl + 스크롤 : 이전 또는 다음 축을 현재 축으로 선택
* 왼쪽 드래그 : 커서/창 너비의 백분율에 따라 슬라이스 선택
* 위/아래 화살표 : 이전 또는 다음 축을 현재 축으로 선택
