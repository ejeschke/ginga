대비 모드는 Ginga 이미지 뷰어에서 이미지의 대비를 조정하는 키 할당을
활성화합니다.

모드 진입 방법
--------------
* 스페이스, 그다음 「t」

모드 종료 방법
--------------
* Esc

모드의 기본 키 할당
-------------------
* T : 대비를 기본값으로 복원
* 왼쪽 드래그 : 대비 조정
  * 컬러맵을 대화식으로 이동/스트레치(대비와 바이어스).
  * 좌우로 이동은 시프트, 상하는 스트레치를 제어.
  * 대비가 만족스러우면 버튼을 놓습니다.
* 오른쪽 클릭 : 대비를 기본값으로 복원
* 스크롤 휠 : 대비 변경(Ctrl을 추가하면 더 세밀하게 조정)
* Shift + 스크롤 휠 : 밝기 변경(Ctrl을 추가하면 더 세밀하게 조정)
* Ctrl + 팬 제스처 : 대비 변경
* Shift + 팬 제스처 : 밝기 변경
