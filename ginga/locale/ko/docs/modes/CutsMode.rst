Cuts 모드는 Ginga 이미지 뷰어에서 이미지의 하한·상한 컷 레벨을 조정하는
키 할당을 활성화합니다.

모드 진입 방법
--------------
* 스페이스, 그다음 「s」

모드 종료 방법
--------------
* Esc

모드의 기본 키 할당
-------------------
* l : 하한 컷 레벨을 커서 아래 픽셀 값으로 설정
* k : 상한 컷 레벨을 커서 아래 픽셀 값으로 설정
* S : 하한·상한 컷 레벨을 이미지의 최소/최대값으로 설정
* A : 하한·상한 컷 레벨을 0, 255로 설정; 주로 표준 RGB 이미지에 유용
* a : 현재 선택된 자동 컷 알고리즘과 매개변수로 자동 컷 수행
* b, 위쪽 화살표 : 목록의 이전 자동 컷 알고리즘 선택
* n, 아래쪽 화살표 : 목록의 다음 자동 컷 알고리즘 선택
* 콜론 : 이 뷰어에서 새 이미지의 자동 컷을 켜기/끄기 전환
* 세미콜론 : 이 뷰어에서 새 이미지의 자동 컷을 「override」로 설정
* 스크롤 휠 : 레벨을 좁히거나 늘려 대비 조정; 한 방향은 좁히고 다른 방향은 늘림
* Ctrl + 스크롤 : 레벨을 좁히거나 늘려 미세 대비 조정; 스크롤과 비슷하나 변화량이 작음
* Shift + 스크롤 : 현재 자동 컷 알고리즘 변경
* 왼쪽 드래그 : 커서를 움직여 레벨 조정; 좌우는 상한, 상하는 하한을 조정
* Shift + 왼쪽 드래그 : 커서를 움직여 하한 조정; 좌우가 하한을 조정
* Ctrl + 왼쪽 드래그 : 커서를 움직여 상한 조정; 좌우가 상한을 조정
* 오른쪽 클릭 : 자동 레벨 수행(「a」와 동일)
* 핀치 제스처 : 하한과 상한 컷 레벨 사이 간격을 넓히거나 좁힘(스크롤 휠과 유사)
* Ctrl + 팬 제스처 : 상한 컷 레벨을 위아래로 변경
* Shift + 팬 제스처 : 하한 컷 레벨을 위아래로 변경
