회전 모드는 Ginga 이미지 뷰어에서 이미지의 축을 뒤집거나 교환하거나 이미지를
회전하는 키 할당을 활성화합니다.

모드 진입 방법
--------------
* 스페이스, 그다음 「r」

모드 종료 방법
--------------
* Esc

모드의 기본 키 할당
-------------------
* 「[」: (전환) 이미지를 X 축으로 뒤집기
* 「{」: X 축 복원
* 「]」: (전환) 이미지를 Y 축으로 뒤집기
* 「}」: Y 축 복원
* 백슬래시 : (전환) X 축과 Y 축 교환
* 「|」: 교환된 축을 원래대로 복원
* R : 회전을 0도로 재설정(뒤집기나 교환은 재설정하지 않음)
* t : 뒤집기나 교환 재설정
* 마침표 : 이미지를 +90도씩 회전
* 쉼표 : 이미지를 -90도씩 회전
* o : 북쪽이 위, 동쪽이 왼쪽이 되도록 이미지 방향 지정
* O : 북쪽이 위, 동쪽이 오른쪽이 되도록 이미지 방향 지정
* 왼쪽 드래그 : 이미지를 대화식으로 회전
* 오른쪽 클릭 : 회전을 0도로 재설정(R과 동일)
* 회전 제스처 : 이미지를 대화식으로 회전
