CMap 모드는 Ginga 이미지 뷰어에서 이미지의 컬러맵과 강도 맵을 조정하는
키 할당을 활성화합니다.

모드 진입 방법
--------------
* 스페이스, 그다음 「y」

모드 종료 방법
--------------
* Esc

모드의 기본 키 할당
-------------------
* Y : 컬러맵을 기본값(회색)으로 재설정
* r : 컬러맵 복원(반전, 회전 또는 스트레치 취소)
* I : 현재 컬러맵 반전(회전이나 스트레치는 유지)
* 위쪽 화살표 : 목록의 이전 컬러맵 설정(회전/스트레치 유지)
* 아래쪽 화살표 : 목록의 다음 컬러맵 설정(회전/스트레치 유지)
* c : 보이는 컬러맵을 이미지 위에 겹쳐 표시
* i : 기본 강도 맵 「ramp」 복원
* j, 왼쪽 화살표 : 목록의 이전 강도 맵 설정
* k, 오른쪽 화살표 : 목록의 다음 강도 맵 설정
* 스크롤 : 목록에서 컬러맵 선택
* Ctrl + 스크롤 : 목록에서 강도 맵 선택
* 팬 제스처 : 목록에서 컬러맵 선택(힌트: 위/아래 화살표 키로 선택 확정)
* 왼쪽 드래그 : 현재 컬러맵 회전
* 오른쪽 클릭 : 컬러맵 복원(「r」과 동일)
