Pan 모드는 Ginga 이미지 뷰어에서 팬 위치(중앙 픽셀 아래의 위치)와 줌 레벨
(스케일)을 설정하는 키 할당을 활성화합니다.

모드 진입 방법
--------------
* 스페이스, 그다음 「q」

모드 종료 방법
--------------
* Esc

모드의 기본 키 할당
-------------------
* 더하기, 등호 : 한 줌 레벨 확대
* 빼기, 밑줄 : 한 줌 레벨 축소
* 1-9,0 : 레벨 N으로 줌(0은 10)
* Shift + 1-9,0 : 레벨 -N으로 줌(0은 -10)
* 백틱 : 이미지를 창 크기에 맞춤
* 큰따옴표 : 이 뷰어에서 새 이미지의 자동 맞춤 켜기/끄기 전환
* 작은따옴표 : 이 뷰어에서 새 이미지의 자동 맞춤을 「override」로 설정
* p : 커서 아래 위치로 팬
* c : 이미지 중심으로 팬
* z : 줌 레벨(스케일) 저장
* r : 커서로 팬하고 저장한 스케일 레벨로 줌
* 왼쪽/오른쪽/위/아래 화살표 : 작은 백분율만큼 팬
* Shift + 화살표 : 매우 작은 백분율만큼 팬
* PageUp(PageDown) : 화면의 큰 백분율만큼 위(아래)로 팬
* Home(End) : 이미지의 위쪽(아래쪽) 방향으로 팬
* 「?」: 이 뷰어에서 새 이미지의 자동 센터링 켜기/끄기 전환
* 「/」: 이 뷰어에서 새 이미지의 자동 센터링을 「override」로 설정
* 스크롤 : 이미지 줌(스케일)
* 왼쪽 드래그 : 뷰를 팬
* 오른쪽 드래그 : 카메라로 뷰를 팬
* 팬 제스처 : 제스처에 비례하여 뷰를 팬
* 핀치 제스처 : 제스처에 비례하여 이미지 줌
