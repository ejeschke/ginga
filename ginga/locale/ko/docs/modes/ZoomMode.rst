Zoom 모드는 Ginga 이미지 뷰어에서 팬 위치(중앙 픽셀)와 줌 레벨(스케일)을
설정하는 키 할당을 활성화합니다.

스크롤과 팬 조작 방식이 Pan 모드와 다릅니다.

모드 진입 방법
--------------
* 스페이스, 그다음 「w」

모드 종료 방법
--------------
* Esc

모드의 기본 키 할당
-------------------
* 가운데 드래그 : 이미지 안을 자유롭게 팬
* 왼쪽 클릭 : 커서 아래로 팬하고 확대
* 오른쪽 클릭 : 축소
* 팬 제스처 : 이미지를 확대/축소
* Shift + 팬 제스처 : 원점을 설정하여 이미지를 확대/축소
