Plot2D 모드는 Matplotlib 플롯 뷰어(ginga.plot.PlotView 모듈)에서 팬 위치와
줌 레벨(스케일)을 설정하는 키 할당을 활성화합니다.

모드 진입 방법
--------------
* 스페이스, 그다음 「p」

모드 종료 방법
--------------
* Esc

모드의 기본 키 할당
-------------------
* Shift + 왼쪽 클릭 : 팬 위치 설정
* 가운데 클릭 : 팬 위치 설정
* 스크롤 : 확대/축소
* Ctrl + 스크롤 : X 축만 확대/축소
* Shift + 스크롤 : Y 축만 확대/축소(MacOS에서는 트랙패드만)
* Alt + 스크롤(마우스) : Y 축만 확대/축소(Mac에서는 Option 키)
* Alt + 스크롤 : 커서 위치에서 확대/축소

키 입력 할당
------------

확대/축소
---------
* 등호 : 한 줌 레벨 확대
* Ctrl + 등호 : X 축을 한 줌 레벨 확대
* 더하기(Shift + 등호) : Y 축을 한 줌 레벨 확대
* 빼기 : 한 줌 레벨 축소
* Ctrl + 빼기 : X 축을 한 줌 레벨 축소
* 밑줄(Shift + 빼기) : Y 축을 한 줌 레벨 축소
* 9 : 커서 위치를 유지하며 축소
* Ctrl + 9 : 커서 위치를 유지하며 X 축 축소
* 왼쪽 괄호(Shift + 9) : 커서 위치를 유지하며 Y 축 축소
* 0 : 커서 위치를 유지하며 확대
* Ctrl + 0 : 커서 위치를 유지하며 X 축 확대
* 오른쪽 괄호(Shift + 0) : 커서 위치를 유지하며 Y 축 확대
* 백틱 : X 축과 Y 축을 창에 맞춤
* 1 : X 축만 창에 맞춤
* 2 : Y 축만 창에 맞춤
* k : X 하한 범위를 커서의 X 값으로 설정
* l : X 상한 범위를 커서의 X 값으로 설정
* K : Y 하한 범위를 커서의 Y 값으로 설정
* L : Y 상한 범위를 커서의 Y 값으로 설정
* g : 격자 켜기/끄기 전환
* f : 범례 켜기/끄기 전환

팬
--
* 왼쪽 화살표 : 왼쪽으로 팬
* 오른쪽 화살표 : 오른쪽으로 팬
* 위쪽 화살표 : 위로 팬
* 아래쪽 화살표 : 아래로 팬
