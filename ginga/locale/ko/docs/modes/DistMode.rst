Dist 모드는 Ginga 이미지 뷰어에서 이미지의 색 분포를 조정하는 키 할당을
활성화합니다.

이 알고리즘들은 「곡선」 유형 프로파일과 유사합니다: 「linear」, 「log」,
「power」, 「sqrt」, 「squared」, 「asinh」, 「sinh」, 「histeq」.

모드 진입 방법
--------------
* 스페이스, 그다음 「d」

모드 종료 방법
--------------
* Esc

모드의 기본 키 할당
-------------------
* D : 색 분포 알고리즘을 「linear」로 재설정
* b, 위쪽 화살표 : 목록의 이전 분포 알고리즘 선택
* n, 아래쪽 화살표 : 목록의 다음 분포 알고리즘 선택
* 스크롤 휠 : 스크롤하여 색 분포 알고리즘 선택
* 팬 제스처 : 스와이프하여 색 분포 알고리즘 선택(힌트: 위/아래 화살표 키로 확정)
