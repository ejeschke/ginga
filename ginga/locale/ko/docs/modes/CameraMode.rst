카메라 모드는 OpenGL 백엔드로 동작하는 Ginga 이미지 뷰어에서 카메라를
조작하는 키 할당을 활성화합니다.

.. note:: 카메라 모드는 뷰어가 OpenGL 백엔드를 사용하지 않으면 작동하지
          않습니다.

모드 진입 방법
--------------
* 스페이스, 그다음 「c」

모드 종료 방법
--------------
* Esc

모드의 기본 키 할당
-------------------
* s : 나중에 복원할 수 있도록 현재 카메라 설정을 저장
* r : 저장한 카메라 설정으로 되돌리기
* 3 : 카메라 뷰를 3D 모드로 전환
* 스크롤 : 카메라 추적(달리 인/아웃)
* 왼쪽 드래그 : 카메라로 뷰를 궤도 회전
* 오른쪽 드래그 : 카메라로 뷰를 팬
