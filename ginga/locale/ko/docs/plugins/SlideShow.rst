이미지 슬라이드 쇼를 재생합니다.

**플러그인 유형: 로컬**

``SlideShow`` 은 로컬 플러그인으로, 채널에 연결되어 있습니다.
싱글턴이 아니므로 각 채널마다 여러 인스턴스를 열 수 있습니다.

**사용법**

***슬라이드 쇼 불러오기***

플러그인을 시작한 후 「불러오기」 버튼을 사용하여 슬라이드 쇼를 불러올 수
있습니다(슬라이드 쇼 파일 형식은 아래 참조). 그런 다음 파일을 외부에서 편집한 후
언제든지 「다시 불러오기」를 눌러 이 슬라이드 쇼를 다시 불러올 수 있습니다.

***슬라이드 쇼 재생***

「이전」과 「다음」 버튼을 사용하여 목록 내에서 수동으로 뒤로 가고 앞으로 갈 수
있습니다. 이 두 버튼 사이의 스핀 버튼 컨트롤은 목록 내의 특정 슬라이드로
이동시킵니다.

「시작」과 「중지」 버튼은 슬라이드 쇼 내에서 자동 진행을 시작하거나 중지하는 데
사용됩니다.

***지속 시간 제어***

각 슬라이드는 다음 슬라이드로 넘어가기까지의 시간을 제어하는 별도의 「duration」
매개변수(초)를 가질 수 있지만, 어떤 슬라이드에 이것이 없으면 기본 지속 시간이
사용됩니다. 기본 지속 시간은 「기본 지속 시간」이라고 표시된 컨트롤을 사용하여
설정할 수 있습니다.

기본 지속 시간 컨트롤 아래에는 슬라이드의 지속 시간과 쇼의 총 지속 시간을
표시하는 레이블이 있습니다.

**슬라이드 쇼 파일 형식**

슬라이드 쇼 파일 형식은 헤더 줄이 있는 쉼표로 구분된(CSV) 일반 텍스트 파일입니다.
파일에는 「file」이라는 제목의 열이 하나 이상 있어야 합니다. 이 열에는 각
슬라이드에서 불러올 파일 경로의 파일 이름(상대 또는 절대)이 들어 있습니다.

***선택 열***

* 「duration」:  각 슬라이드의 지속 시간(초)을 포함해야 함
* 「position」: 쇼에서 슬라이드의 위치를 나타냄.
  슬라이드 쇼 파일을 편집할 때 슬라이드 재정렬을 더 쉽게 하기 위해 부동 소수점
  숫자를 사용할 수 있습니다.
