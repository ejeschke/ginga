
``Info`` 플러그인은 포커스된 채널 이미지에 대해 흔히 유용한 메타데이터 창을
제공합니다. 일반적인 정보에는 일부 메타데이터 헤더 값, 좌표, 이미지의 치수, 최솟값과
최댓값 등이 포함됩니다. 커서를 이미지 위에서 움직이면 X, Y, Value, RA, DEC 값이
업데이트되어 커서 아래의 값을 반영합니다.

**플러그인 유형: 전역**

``Info`` 는 전역 플러그인입니다. 하나의 인스턴스만 열 수 있습니다.

**사용법**

``Info`` 인터페이스 하단에는 색상 분포와 컷 레벨 컨트롤이 있습니다. 컷 레벨 상자
위의 선택기를 사용하면 이미지의 값을 색상 맵에 매핑하는 여러 분포 알고리즘 중에서
선택할 수 있습니다. 선택지는 「linear」, 「log」, 「power」, 「sqrt」, 「squared」,
「asinh」, 「sinh」, 「histeq」(히스토그램 평활화)입니다.

그 아래에 낮은 및 높은 컷 레벨이 표시되고 조정할 수 있습니다. 「Auto Levels」 버튼을
누르면 현재 자동 컷 레벨 알고리즘과 채널 기본 설정에 정의된 매개변수를 기반으로 컷
레벨이 다시 계산됩니다.

「Auto Levels」 버튼 아래에는 현재 활성 채널에 대한 「Cut New」, 「Zoom New」,
「Center New」 설정의 상태가 표시됩니다. 이들은 채널에 추가되는 새 이미지가 자동 컷
레벨, 창 맞춤, 이미지 중앙으로의 팬에 의해 어떻게 영향을 받는지 나타냅니다.

「Follow New」 체크박스는 뷰어가 채널에 추가된 새 이미지를 자동으로 표시할지 제어
합니다. 「Raise New」 체크박스는 새 이미지가 추가될 때 이미지 뷰어 창을 앞으로
올릴지 제어합니다. 이 두 컨트롤은 예를 들어 외부 프로그램이 뷰어에 이미지를 추가하고
있고 특정 이미지를 검사하는 작업의 중단을 방지하고 싶은 경우에 유용할 수 있습니다.

전역 플러그인으로서 ``Info`` 는 새 채널로의 포커스 변경에 응답하여 새 채널의
메타데이터를 표시합니다. 일반적으로 사용자 인터페이스의 「Synopsis」 탭 아래에
나타납니다.

이 플러그인은 보통 닫을 수 있도록 구성되어 있지 않지만, 사용자가 구성 파일에서
「closeable」 설정을 True로 설정하여 그렇게 만들 수 있습니다. 그러면 UI 하단에 닫기
및 도움말 버튼이 추가됩니다.
