
Ginga 참조 뷰어를 위한 다운로드 GUI.

**플러그인 유형: 전역**

``Download`` 는 전역 플러그인입니다. 하나의 인스턴스만 열 수 있습니다.

**사용법**

이 플러그인을 열어 URI 다운로드의 진행 상황을 모니터링하십시오. 「Plugins」 또는
「Operations」 메뉴를 사용하여 시작하고 「Util」 범주에서 「Downloads」 플러그인을
선택하십시오.

다운로드를 시작하려면 URI를 채널 이미지 뷰어나 ``Thumbs`` 창으로 끌어다 놓기만
하면 됩니다.

각 항목의 「지우기」 버튼을 클릭하여 언제든지 다운로드에 대한 정보를 제거할 수
있습니다. 하단의 「모두 지우기」 버튼을 클릭하여 모든 다운로드의 항목을 지울 수
있습니다.

현재 진행 중인 다운로드를 취소하는 것은 불가능합니다.

**설정**

``auto_clear_download`` 옵션을 `True` 로 설정하면 다운로드가 완료될 때 다운로드
항목이 창에서 자동으로 삭제됩니다. 다운로드된 파일은 제거하지 않습니다.

다운로드 폴더는 ~/.ginga/general.cfg 의 「download_folder」 설정에 값을 지정하여
사용자가 정의할 수 있습니다. 지정되지 않으면 기본적으로 플랫폼별 기본 임시
디렉터리 내의 폴더가 됩니다(Python 「tempfile」 모듈이 알려주는 대로).
