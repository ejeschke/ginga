``Header`` 플러그인은 이미지와 연결된 메타데이터의 목록을 제공합니다.

**플러그인 유형: 전역**

``Header`` 는 전역 플러그인입니다. 하나의 인스턴스만 열 수 있습니다.

**사용법**

``Header`` 플러그인은 이미지의 FITS 키워드 메타데이터를 표시합니다.
처음에는 기본 HDU 메타데이터만 표시됩니다. 그러나 ``MultiDim`` 플러그인과
함께 사용하면 다른 HDU의 메타데이터도 표시됩니다. 자세한 내용은 ``MultiDim``
을 참조하십시오.

UI 왼쪽 아래의 「정렬 가능」 체크박스가 선택되어 있으면 열 머리글을 클릭할
때 해당 열의 값으로 테이블이 정렬되며, 이는 특정 키워드를 빠르게 찾는 데
유용할 수 있습니다.

「기본 헤더 포함」 체크박스는 기본 HDU 키워드의 포함 여부를 전환합니다. 이미지가
기본 헤더를 저장하지 않는 옵션으로 만들어진 경우 이 옵션은 비활성화될 수
있습니다.
