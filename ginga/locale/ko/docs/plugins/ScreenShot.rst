채널 뷰어 이미지를 PNG 또는 JPEG 이미지로 캡처합니다.

**사용법**

1. 「유형」 콤보 상자에서 스냅의 RGB 그래픽 유형을 선택합니다.
2. 캡처하려는 대로 채널 이미지가 되면 「스냅」을 누릅니다.

RGB 이미지의 사본이 ``ScreenShot`` 뷰어에 불러와집니다.
일반 Ginga 뷰어처럼 ``ScreenShot`` 뷰어 안에서 팬하고 확대/축소하여 세부 사항을
살펴볼 수 있습니다(예: JPEG와 PNG 형식의 확대된 차이 보기).

3. 원하는 이미지가 될 때까지 (1)과 (2)를 반복합니다.
4. 「폴더」 텍스트 상자에 새 파일의 유효한 경로를 입력합니다.
5. 「이름」 텍스트 상자에 새 파일의 유효한 이름을 입력합니다.
   파일 확장자를 추가할 필요는 없습니다. 필요하면 추가됩니다.
6. 「저장」 버튼을 누릅니다. 지정한 위치에 파일이 저장됩니다.

**참고**

* PNG는 겹친 그래픽의 아티팩트가 더 적지만, 파일이 JPEG보다 큽니다.
* 「가운데 정렬」 버튼은 스냅 이미지를 가운데에 배치하고, 「맞춤」은 창에 맞게
  줌을 설정하며, 「지우기」는 이미지를 지웁니다. 「전체」를 누르면 100 % 픽셀
  (1:1 배율)로 확대합니다.
* 「화면 크기」 체크박스(기본적으로 선택됨)는 이미지를 채널 뷰어 창과 정확히
  같은 크기로 저장합니다. 다른 크기로 저장하려면 이 상자의 선택을 해제하고
  「너비」와 「높이」 상자로 크기를 설정하십시오.
* 「가로세로 비율 고정」 기능은 「화면 크기」가 해제된 경우에만 작동합니다.
  활성화된 경우 너비나 높이를 변경하면 「비율」 상자에 표시된 가로세로 비율을
  유지하기 위해 다른 매개변수가 변경됩니다.
