``Zoom`` 플러그인은 연결된 채널 이미지에서 커서 위치 아래를 중심으로 한 잘라낸
영역의 확대 이미지를 보여줍니다. 커서를 이미지 위에서 움직이면 줌 이미지가
업데이트되어 픽셀을 자세히 검사하거나 다른 플러그인 작업과 함께 정밀하게 제어할
수 있습니다.

**플러그인 유형: 전역**

``Zoom`` 은 전역 플러그인입니다. 하나의 인스턴스만 열 수 있습니다.

**사용법**

줌 창의 배율은 「확대 정도」 슬라이더를 조정하여 변경할 수 있습니다.

두 가지 작동 모드가 가능합니다 -- 절대 줌과 상대 줌:

* 절대 모드에서는 확대 정도가 잘라낸 영역에 표시되는 줌 수준을 정확하게
  제어합니다. 예를 들어, 채널 이미지가 10배로 확대되어 있어도 확대 정도가 3배로
  설정되어 있으면 줌 이미지는 3배 이미지만 보여줍니다.

* 상대 모드에서는 확대 정도 설정이 채널 이미지의 줌 설정에 상대적인 것으로
  해석됩니다. 확대 정도가 3배로 설정되고 채널 이미지가 10배로 확대되어 있으면
  표시되는 줌 이미지는 13배(10배 + 3배)가 됩니다. 확대 정도 설정은 1 미만일 수도
  있으므로, 채널 이미지가 3배 줌일 때 1/3배 설정이면 1배 줌 이미지가 생성됩니다.

「새로 고침 간격」 설정은 ``Zoom`` 플러그인이 커서의 움직임에 반응하여 줌 이미지를
업데이트하는 속도를 제어합니다. 값은 밀리초 단위로 지정됩니다.

.. tip:: 보통 작은 새로 고침 간격을 설정하면 줌 이미지의 전반적인 반응성이
         *향상*되며, 기본값 20이 적당합니다. 줌 이미지가 너무 끊기거나 채널 이미지
         창에서의 마우스 움직임과 동기화되지 않은 것처럼 보이면 값을 여러 가지로
         시험해 볼 수 있습니다.

「기본값」 버튼은 컨트롤의 기본 설정을 복원합니다.

``~/.ginga/plugin_Zoom.cfg`` 를 사용하여 사용자 지정할 수 있으며, 여기서 ``~`` 는
사용자의 HOME 디렉터리입니다:

.. code-block:: Python

  #
  # Zoom plugin preferences file
  #
  # Place this in file under ~/.ginga with the name "plugin_Zoom.cfg"

  # default zoom level
  zoom_amount = 3

  # refresh interval (sec)
  # NOTE: usually a small delay speeds things up
  refresh_interval = 0.02
