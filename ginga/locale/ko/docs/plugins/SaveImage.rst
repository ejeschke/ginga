이미지를 출력 파일로 저장합니다.

**플러그인 유형: 전역**

``SaveImage`` 는 전역 플러그인입니다. 하나의 인스턴스만 열 수 있습니다.

**사용법**

이 전역 플러그인은 Ginga에서 가한 변경 사항을 출력 이미지로 다시 저장하는 데
사용됩니다. 예를 들어 ``Mosaic`` 플러그인으로 만든 모자이크 이미지 등입니다.
현재는 FITS 이미지(단일 또는 다중 확장)만 지원됩니다.

출력 디렉터리(예: ``/mypath/outputs/``), 접미사(예: ``ginga``), 이미지
채널(``Image``), 선택한 이미지(예: ``image1.fits``)가 주어지면 출력 파일은
``/mypath/outputs/image1_ginga_Image.fits`` 가 됩니다. 채널 이름의 포함 여부는
선택 사항이며 플러그인 구성 파일 ``plugin_SaveImage.cfg`` 를 사용하여 생략할 수
있습니다.
수정된 확장에는 Ginga에서 추출한 새 헤더나 데이터가 들어가고, 수정되지 않은
확장은 그대로 유지됩니다. 전역 플러그인 ``ChangeHistory`` 의 관련 변경 로그
항목이 해당 ``PRIMARY`` 헤더의 기록에 삽입됩니다.

.. note:: 이 플러그인은 구성 파일 ``general.cfg`` 에서 ``FITSpkg`` 에 무엇이
          선택되어 있든 관계없이, 출력 이미지를 쓰는 데 ``astropy.io.fits``
          모듈을 사용합니다.
