큐브를 이등분하는 직선을 따라 픽셀 값을 그래프로 나타내는 플러그인입니다.

**플러그인 유형: 로컬**

``LineProfile`` 은 로컬 플러그인으로, 채널에 연결되어 있습니다.
각 채널마다 하나의 인스턴스를 열 수 있습니다.

**사용법**

.. warning::

   어떤 축을 선택할 수 있는지에 대한 제한이 없습니다.
   따라서 플롯이 무의미할 수 있습니다.

``LineProfile`` 플러그인은 다차원(즉, 3D 이상) 이미지에 사용됩니다. 선택한 축을
따라 현재 커서 위치의 픽셀 값을 플롯합니다. 또는 영역이 선택된 경우 각 프레임의
평균을 플롯합니다. 이는 일반적인 스펙트럼 선 프로파일을 만드는 데 사용할 수
있습니다. 현재 표시된 프레임의 데이터 점에 마커가 놓입니다.

표시된 X축은 FITS 헤더의 ``CRVAL*``, ``CDELT*``, ``CRPIX*``, ``CTYPE*``,
``CUNIT*`` 키워드를 사용하여 구성됩니다. 키워드 중 하나라도 사용할 수 없으면
축은 대신 ``NAXIS*`` 값으로 대체됩니다.

표시된 Y축은 ``BTYPE`` 와 ``BUNIT`` 를 사용하여 구성됩니다. 이들이 사용 불가능한
경우 단순히 픽셀 값을 「Signal」 로 레이블합니다.

이 플러그인 사용 방법:

1. 축을 선택합니다.
2. 커서로 점을 선택하거나 영역을 그립니다.
3. 해당하는 경우 ``MultiDim`` 을 사용하여 축의 스텝 값을 변경합니다.
