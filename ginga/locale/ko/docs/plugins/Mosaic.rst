합성 이미지를 구성하여 이미지 모자이크를 만드는 플러그인입니다.

**플러그인 유형: 로컬**

``Mosaic`` 은 로컬 플러그인으로, 채널에 연결되어 있습니다.
각 채널마다 하나의 인스턴스를 열 수 있습니다.

**사용법**

.. warning:: 이는 메모리를 매우 많이 사용할 수 있습니다.

이 플러그인은 사용자가 제공한 이미지(예: ``FBrowser`` 사용)를 이용하여 채널에
모자이크 이미지를 자동으로 구성하는 데 사용됩니다.
모자이크에서 이미지의 위치는 왜곡 보정 없이 해당 WCS에 의해 결정됩니다. 이는
빠른 미리 보기 도구이며, 이미지 왜곡 등을 고려하는 이미지 드리즐링의 대체가
아닙니다. 모자이크는 메모리에만 존재하지만, ``SaveImage`` 를 사용하여 FITS
파일로 저장할 수 있습니다.

모자이크가 메모리에서 벗어나면 Ginga에서 더 이상 접근할 수 없습니다. 이를
방지하려면 Ginga 데이터 캐시가 충분히 크도록 세션을 구성해야 합니다(설명서의
「Customizing Ginga」 참조).

새 모자이크를 만들려면 FOV를 설정하고 파일을 표시 창으로 끌어다 놓으십시오.
이미지에는 작동하는 WCS가 있어야 합니다. 첫 번째 이미지의 WCS가 다른 타일의
방향을 지정하는 데 사용됩니다.

**`Collage` 플러그인과의 차이점**

- 모든 모자이크 내용을 담기 위해 하나의 큰 배열을 할당함
- 구성은 느리지만, 큰 결과 이미지를 조작하는 것은 더 빠를 수 있음
- 모자이크를 새 데이터 파일로 저장할 수 있음
- 타일 사이의 값을 채움 값(`NaN` 가능)으로 채움
