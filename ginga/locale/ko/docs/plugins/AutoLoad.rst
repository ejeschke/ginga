``AutoLoad`` 는 폴더에서 새 파일을 감시하고, 파일이 나타나면 자동으로 채널에
불러오는 간단한 플러그인입니다.

**플러그인 유형: 로컬**

``AutoLoad`` 는 로컬 플러그인으로, 채널에 연결되어 있습니다.
각 채널마다 하나의 인스턴스를 열 수 있습니다.

.. note:: 이 플러그인을 사용하려면 Python 「watchdog」 패키지를 설치해야
          합니다.

**사용법**

* 감시할 폴더를 설정하려면 「감시 폴더」 필드에 폴더(디렉터리) 경로를 입력하고
  ENTER를 누르거나 「설정」을 클릭하십시오.
* 이 폴더에 추가될 파일을 구별해야 하는 경우, 「정규식」 상자에 Python 정규식을
  입력하고 「설정」을 클릭할 수 있습니다. 이름이 패턴과 일치하는 파일만
  고려됩니다. 정규식은 파일 이름에만 적용되며 폴더 경로의 어느 부분에도 적용되지
  않는다는 점에 유의하십시오.
* 자동 불러오기를 일시 정지하려면 「자동 불러오기 일시 정지」로 표시된 상자를
  선택할 수 있으며, 이는 모든 자동 불러오기를 중지합니다. 이후 상자의 선택을
  해제하면 그 사이에 도착한 파일은 불러오지 않는다는 점에 유의하십시오.

.. note:: 네트워크 드라이브에 있는 폴더의 감시는 작동할 수도, 작동하지 않을
          수도 있습니다.
