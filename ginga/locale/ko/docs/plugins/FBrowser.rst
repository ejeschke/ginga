로컬 파일 시스템을 탐색하고 파일을 불러오는 플러그인입니다.

**플러그인 유형: 전역 또는 로컬**

``FBrowser`` 는 전역/로컬 하이브리드 플러그인으로, 두 방식 중 어느 것으로든
호출할 수 있습니다. 로컬 플러그인으로 호출하면 채널에 연결되며 각 채널마다
하나의 인스턴스를 열 수 있습니다. 전역 플러그인으로 열 수도 있습니다.

**사용법**

불러올 파일의 위치에 도달할 때까지 디렉터리 트리를 탐색하십시오. 파일을 더블
클릭하면 연결된 채널에 불러오고, 파일을 채널 뷰어 창으로 끌어다 놓으면 임의의
채널 뷰어에 불러옵니다.

``Ctrl``(Mac에서는 ``Command``)를 누른 채로 여러 파일을 선택하거나,
``Shift`` 를 누른 채 클릭하여 연속된 범위의 파일을 선택할 수 있습니다.

원하는 이미지의 전체 경로를 텍스트 상자에 입력할 수도 있습니다. 예를 들어
``/my/path/to/image.fits``, ``/my/path/to/image.fits[ext]``,
``/my/path/to/image*.fits[extname,*]`` 와 같이.

로컬 플러그인이므로 ``FBrowser`` 는 닫았다가 다시 시작해도 마지막 디렉터리를
기억합니다.
