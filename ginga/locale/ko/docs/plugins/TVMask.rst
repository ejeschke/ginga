
파일에서 마스크(비대화형 모드)를 이미지 위에 표시합니다.

**플러그인 유형: 로컬**

``TVMask`` 은 로컬 플러그인으로, 채널에 연결되어 있습니다.
각 채널마다 하나의 인스턴스를 열 수 있습니다.

**사용법**

이 플러그인은 FITS 파일을 읽어 마스크를 비대화형으로 표시할 수 있게 하며, 0이
아닌 값은 마스크된 데이터로 가정합니다.

서로 다른 마스크를 표시하려면(예: 위에 표시된 것처럼 일부는 녹색, 일부는
분홍색으로 마스크):

1. 드롭다운 메뉴에서 녹색을 선택하십시오. 또는 원하는 알파 값을 입력하십시오.
2. 「마스크 불러오기」 버튼을 사용하여 관련 FITS 파일을 불러오십시오.
3. (1)을 반복하되 이번에는 드롭다운 메뉴에서 분홍색을 선택하십시오.
4. (2)를 반복하되 다른 FITS 파일을 선택하십시오.
5. 세 번째 마스크도 분홍색으로 표시하려면 드롭다운 메뉴를 변경하지 않고 (4)를
   반복하십시오.

표 목록에서 항목(또는 여러 항목)을 선택하면 이미지의 마스크가 강조 표시됩니다.
강조 표시에는 미리 정의된 색상과 알파가 사용됩니다(아래에서 사용자 지정 가능).

이 플러그인이 활성 상태일 때 이미지에 사각형을 그려서 이미지와 표 목록 모두에서
어떤 영역 내의 모든 마스크를 강조 표시할 수도 있습니다.

「숨기기」 버튼을 누르면 마스크가 숨겨지지만 플러그인의 메모리는 지워지지
않습니다. 즉, 「표시」를 누르면 같은 마스크가 같은 이미지에 다시 나타납니다.
그러나 「잊기」를 누르면 마스크가 표시와 메모리 모두에서 지워집니다. 즉, 마스크를
다시 만들려면 파일을 다시 불러와야 합니다.

같은 마스크를 다른 색상이나 알파로 다시 그리려면 「잊기」를 누르고 필요에 따라
위 단계를 반복하십시오.

지향/차원이 매우 다른 이미지가 같은 채널에 표시되는 경우, 한 이미지에 속하지만
다른 이미지의 밖으로 벗어나는 마스크는 후자에 나타나지 않습니다.

이 플러그인이 읽을 수 있는 마스크를 만들려면 ``Drawing`` 플러그인의 결과를 사용할
수 있습니다(그린 후 「마스크 생성」을 누르고 ``SaveImage`` 를 사용하여 마스크
저장). 그 외에 ``astropy.io.fits`` 등을 사용하여 손으로 FITS 파일을 만들 수도
있습니다.

``TVMark`` 와 함께 사용하면 Ginga에서 점원과 마스크된 영역을 모두 겹쳐 표시할 수
있습니다.

``~/.ginga/plugin_TVMask.cfg`` 를 사용하여 사용자 지정할 수 있으며, 여기서 ``~``
는 사용자의 HOME 디렉터리입니다:

.. code-block:: Python

  #
  # TVMask plugin preferences file
  #
  # Place this in file under ~/.ginga with the name "plugin_TVMask.cfg"

  # Mask color -- Any color name accepted by Ginga
  maskcolor = 'green'

  # Mask alpha (transparency) -- 0=transparent, 1=opaque
  maskalpha = 0.5

  # Highlighted mask color and alpha
  hlcolor = 'white'
  hlalpha = 1.0
