``Blink`` 은 사용자가 선택한 속도로 채널에 표시된 이미지 사이를 전환합니다.
또는 기본 작업 공간의 채널 사이를 전환할 수도 있습니다. 두 경우 모두 주요
목적은 눈을 깜빡이듯 짧은 시간 안에 (채널 내에서 또는 채널 간에) 이미지를
시각적으로 비교하고 대조하는 것입니다.

**플러그인 유형: 로컬 또는 전역**

``Blink`` 은 로컬 플러그인으로 호출하면 채널의 이미지를 순환하고, 전역
플러그인으로 호출하면 채널을 순환합니다.

로컬 플러그인은 「작업」 버튼에서, 전역 플러그인은 「플러그인」 메뉴에서
시작합니다.

**사용법**

이미지 전환 사이의 간격을 「간격」 상자에 초 단위로 설정하십시오. 그런 다음
「깜빡임 시작」을 눌러 시간 지정 순환을 시작하고 「깜빡임 중지」를 눌러
멈추십시오.

「간격」의 숫자를 변경하고 ``Enter`` 를 누르면 순환이 실행되는 동안 순환 시간을
동적으로 변경할 수 있습니다.
