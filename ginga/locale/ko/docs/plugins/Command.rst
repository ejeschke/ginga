이 플러그인은 참조 뷰어를 위한 명령줄 인터페이스를 제공합니다.

.. note:: 이 명령줄은 플러그인 UI *내부*에서 사용하기 위한 것입니다. *원격*
          명령줄 인터페이스를 찾고 있다면 ``RC`` 플러그인을 참조하십시오.

**플러그인 유형: 전역**

``Command`` 는 전역 플러그인입니다. 하나의 인스턴스만 열 수 있습니다.

**사용법**

명령과 매개변수 목록 가져오기::

        g> help

셸 명령 실행::

        g> !cmd arg arg ...

**참고**

특히 강력한 도구는 ``reload_local`` 및 ``reload_global`` 명령을 사용하여 개발
중인 플러그인을 다시 불러오는 것입니다. 이렇게 하면 참조 뷰어를 다시 시작하고
힘들게 데이터 등을 다시 불러올 필요가 없습니다. 플러그인을 닫고, 적절한
「reload」 명령을 실행한 다음(도움말을 참조!) 플러그인을 다시 시작하기만 하면
됩니다.

.. note:: 플러그인 자체가 *아닌* 다른 모듈을 수정한 경우, 이 명령들로는 다시
          불러오지 않습니다.
