``Pan`` 플러그인은 마지막으로 포커스를 가졌던 채널 이미지 전체의 「조감도」
보기를 제공하는 작은 팬 이미지를 제공합니다. 채널 이미지가 2배 이상 확대된
경우, 팬 영역이 ``Pan`` 이미지에 사각형으로 그래픽으로 표시됩니다.

**플러그인 유형: 로컬**

``Pan`` 은 전역 플러그인입니다. 하나의 인스턴스만 열 수 있습니다.

**사용법**

채널 이미지는 클릭하거나 드래그하여 사각형을 배치함으로써 팬할 수 있습니다.
마우스 오른쪽 버튼으로 사각형을 드래그하면 채널 이미지 뷰어가 해당 영역에
맞추려고 시도합니다(그린 사각형과 창 크기 사이의 가로세로 비율 차이를 고려).
``Pan`` 이미지에서 스크롤하면 채널 이미지가 확대/축소됩니다.

``Pan`` 이미지의 색/강도 맵과 컷 레벨은 해당 채널 이미지에서 변경될 때
업데이트됩니다.
``Pan`` 이미지는 채널에서 보고 있는 FITS HDU에 유효한 WCS 메타데이터가 있는
경우 세계 좌표계(WCS) 나침반도 표시합니다.

``Pan`` 플러그인은 보통 「Info」 탭 아래, ``Info`` 플러그인 옆에 하위 창으로
나타납니다.

이 플러그인은 보통 닫을 수 있도록 구성되어 있지 않지만, 사용자는 구성 파일에서
「closeable」 설정을 True로 설정하여 닫을 수 있게 만들 수 있습니다. 그러면 UI
하단에 닫기 및 도움말 버튼이 추가됩니다.
