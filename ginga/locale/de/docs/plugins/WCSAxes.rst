Ein Plugin zum Erzeugen einer WCS-Achsenüberlagerung im geladenen Bild.

**Plugin-Typ: Lokal**

``WCSAxes`` ist ein lokales Plugin, das heißt, es ist einem Kanal zugeordnet.
Für jeden Kanal kann eine Instanz geöffnet werden.

**Verwendung**

Solange das Bild ein gültiges WCS besitzt, werden WCS-Achsen angezeigt.
Verwenden Sie die Plugin-Oberfläche oder die Konfigurationsdatei, um die
Achsenanzeige anzupassen.
