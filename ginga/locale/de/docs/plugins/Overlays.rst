Ein Plugin zum Erzeugen von Farbüberlagerungen, die Unter- und Überbelichtung
im geladenen Bild darstellen.

**Plugin-Typ: Lokal**

``Overlays`` ist ein lokales Plugin, das heißt, es ist einem Kanal zugeordnet.
Für jeden Kanal kann eine Instanz geöffnet werden.

**Verwendung**

Wählen Sie in den Auswahlmenüs Farben für die untere und/oder obere Grenze
(„Farbe unten“ bzw. „Farbe oben“).  Geben Sie die Grenzen für niedrige und
hohe Werte in den Grenzfeldern an („Grenze unten“ bzw. „Grenze oben“).  Legen
Sie die Deckkraft der Überlagerungen mit einem Wert zwischen 0 und 1 im Feld
„Deckkraft“ fest.  Drücken Sie schließlich die Schaltfläche „Wiederholen“.

Die Farbüberlagerung sollte Bereiche unterhalb der unteren Grenze in der
unteren Farbe und Bereiche oberhalb der oberen Grenze in der oberen Farbe
zeigen.  Lassen Sie eine Grenze weg (Feld leer), so wird diese Farbe in der
Überlagerung nicht angezeigt.

Wird ein neues Bild für den Kanal ausgewählt, wird das Überlagerungsbild anhand
der aktuellen Parameter mit den neuen Daten neu berechnet.
