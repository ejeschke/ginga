PNG- oder JPEG-Bilder des Kanalbetrachter-Bildes aufnehmen.

**Verwendung**

1. Wählen Sie im Kombinationsfeld „Typ“ den RGB-Grafiktyp für den Schnappschuss.
2. Drücken Sie „Schnappschuss“, wenn das Kanalbild so aussieht, wie Sie es
   aufnehmen möchten.

Eine Kopie des RGB-Bildes wird in den ``ScreenShot``-Betrachter geladen.
Sie können im ``ScreenShot``-Betrachter wie in einem normalen Ginga-Betrachter
verschieben und zoomen, um Details zu untersuchen (z. B. den vergrößerten
Unterschied zwischen JPEG- und PNG-Format sehen).

3. Wiederholen Sie (1) und (2), bis Sie das gewünschte Bild haben.
4. Geben Sie im Textfeld „Ordner“ einen gültigen Pfad für eine neue Datei ein.
5. Geben Sie im Textfeld „Name“ einen gültigen Namen für eine neue Datei ein.
   Die Dateiendung muss nicht angegeben werden; sie wird bei Bedarf hinzugefügt.
6. Drücken Sie die Schaltfläche „Speichern“.  Die Datei wird am angegebenen Ort
   gespeichert.

**Hinweise**

* PNG bietet weniger Artefakte für überlagerte Grafiken, doch die Dateien sind
  größer als JPEG.
* Die Schaltfläche „Zentrieren“ zentriert das Schnappschussbild; „Anpassen“
  stellt den Zoom so ein, dass es ins Fenster passt; und „Löschen“ löscht das
  Bild.  Drücken Sie „Voll“, um auf 100 % Pixel (1:1-Skalierung) zu zoomen.
* Das Kontrollkästchen „Bildschirmgröße“ (standardmäßig aktiviert) speichert das
  Bild genau in der Größe des Kanalbetrachter-Fensters.  Um in einer anderen
  Größe zu speichern, deaktivieren Sie dieses Kästchen und legen die Größe über
  die Felder „Breite“ und „Höhe“ fest.
* Die Funktion „Seitenverhältnis sperren“ funktioniert nur, wenn
  „Bildschirmgröße“ deaktiviert ist; ist sie aktiviert, so ändert das Ändern von
  Breite oder Höhe den jeweils anderen Parameter, um das im Feld
  „Seitenverhältnis“ angezeigte Verhältnis beizubehalten.
