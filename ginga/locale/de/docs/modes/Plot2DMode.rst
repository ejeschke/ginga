Der Plot2D-Modus aktiviert Tastenbelegungen, mit denen sich die
Verschiebeposition und die Zoomstufe (Skalierung) in einem
Matplotlib-Plot-Betrachter (Modul ginga.plot.PlotView) einstellen lassen.

Modus betreten durch
--------------------
* Leertaste, dann „p“

Modus verlassen durch
---------------------
* Esc

Standardtastenbelegung im Modus
-------------------------------
* Umschalt + Linksklick : Verschiebeposition setzen
* Mittelklick : Verschiebeposition setzen
* Scrollen : hinein-/herauszoomen
* Strg + Scrollen : nur die X-Achse hinein-/herauszoomen
* Umschalt + Scrollen : nur die Y-Achse hinein-/herauszoomen (auf MacOS nur
  Trackpad)
* Alt + Scrollen (Maus) : nur die Y-Achse hinein-/herauszoomen (Option-Taste
  auf Macs)
* Alt + Scrollen : am Cursor hinein-/herauszoomen

Tastenbelegungen
----------------

Zoomen
------
* Gleichheitszeichen : eine Zoomstufe hineinzoomen
* Strg + Gleichheitszeichen : X-Achse eine Zoomstufe hineinzoomen
* Plus (Umschalt + Gleichheitszeichen) : Y-Achse eine Zoomstufe hineinzoomen
* Minus : eine Zoomstufe herauszoomen
* Strg + Minus : X-Achse eine Zoomstufe herauszoomen
* Unterstrich (Umschalt + Minus) : Y-Achse eine Zoomstufe herauszoomen
* 9 : herauszoomen und die Cursorposition beibehalten
* Strg + 9 : X-Achse herauszoomen und die Cursorposition beibehalten
* linke Klammer (Umschalt + 9) : Y-Achse herauszoomen, Cursorposition behalten
* 0 : hineinzoomen und die Cursorposition beibehalten
* Strg + 0 : X-Achse hineinzoomen und die Cursorposition beibehalten
* rechte Klammer (Umschalt + 0) : Y-Achse hineinzoomen, Cursorposition behalten
* Backtick : X- und Y-Achse an das Fenster anpassen
* 1 : nur die X-Achse an das Fenster anpassen
* 2 : nur die Y-Achse an das Fenster anpassen
* k : unteren X-Bereich auf den X-Wert am Cursor setzen
* l : oberen X-Bereich auf den X-Wert am Cursor setzen
* K : unteren Y-Bereich auf den Y-Wert am Cursor setzen
* L : oberen Y-Bereich auf den Y-Wert am Cursor setzen
* g : Gitter ein-/ausschalten
* f : Legende ein-/ausschalten

Verschieben
-----------
* Pfeil links : nach links verschieben
* Pfeil rechts : nach rechts verschieben
* Pfeil hoch : nach oben verschieben
* Pfeil runter : nach unten verschieben
