Der Kontrastmodus aktiviert Tastenbelegungen, mit denen sich der Kontrast
eines Bildes in einem Ginga-Bildbetrachter anpassen lässt.

Modus betreten durch
--------------------
* Leertaste, dann „t“

Modus verlassen durch
---------------------
* Esc

Standardtastenbelegung im Modus
-------------------------------
* T : Kontrast auf die Standardwerte zurücksetzen
* Ziehen mit links : Kontrast anpassen
  * Farbkarte interaktiv verschieben/strecken (Kontrast und Bias).
  * Links/rechts steuert die Verschiebung, hoch/runter die Streckung.
  * Taste loslassen, wenn der Kontrast passt.
* Rechtsklick : Kontrast auf die Standardwerte zurücksetzen
* Scrollrad : Kontrast ändern (mit Strg feiner einstellen)
* Umschalt + Scrollrad : Helligkeit ändern (mit Strg feiner einstellen)
* Strg + Verschiebegeste : Kontrast ändern
* Umschalt + Verschiebegeste : Helligkeit ändern
