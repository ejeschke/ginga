Der Pan-Modus aktiviert Tastenbelegungen, mit denen sich in einem
Ginga-Bildbetrachter die Verschiebeposition (die Position unter dem mittleren
Pixel) und die Zoomstufe (Skalierung) einstellen lassen.

Modus betreten durch
--------------------
* Leertaste, dann „q“

Modus verlassen durch
---------------------
* Esc

Standardtastenbelegung im Modus
-------------------------------
* Plus, Gleichheitszeichen : eine Zoomstufe hineinzoomen
* Minus, Unterstrich : eine Zoomstufe herauszoomen
* 1-9,0 : auf Stufe N zoomen (0 ist 10)
* Umschalt + 1-9,0 : auf Stufe -N zoomen (0 ist -10)
* Backtick : Bild an die Fenstergröße anpassen
* Anführungszeichen : Auto-Anpassung für neue Bilder ein-/ausschalten
* Apostroph : Auto-Anpassung für neue Bilder auf „override“ setzen
* p : zur Position unter dem Cursor verschieben
* c : zur Bildmitte verschieben
* z : Zoomstufe (Skalierung) speichern
* r : zum Cursor verschieben und auf die gespeicherte Skalierung zoomen
* Pfeil links/rechts/hoch/runter : um einen kleinen Prozentsatz verschieben
* Umschalt + Pfeiltasten : um einen sehr kleinen Prozentsatz verschieben
* Bild auf (ab) : um einen großen Prozentsatz des Bildschirms nach oben (unten)
  verschieben
* Pos1 (Ende) : zum oberen (unteren) Rand des Bildes verschieben
* „?“ : Auto-Zentrierung für neue Bilder ein-/ausschalten
* „/“ : Auto-Zentrierung für neue Bilder auf „override“ setzen
* Scrollen : das Bild zoomen (skalieren)
* Ziehen mit links : die Ansicht verschieben
* Ziehen mit rechts : die Ansicht mit der Kamera verschieben
* Verschiebegeste : die Ansicht proportional zur Geste verschieben
* Pinch-Geste : das Bild proportional zur Geste zoomen
