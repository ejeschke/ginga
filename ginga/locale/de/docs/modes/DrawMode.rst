Der Zeichenmodus aktiviert Tastenbelegungen, die das Zeichnen erleichtern.

Modus betreten durch
--------------------
* Leertaste, dann „b“

Modus verlassen durch
---------------------
* Esc

Standardtastenbelegung im Modus
-------------------------------
* v : beim Zeichnen oder Bearbeiten einer Leinwand einem Polygon einen
  Eckpunkt hinzufügen
* z : beim Zeichnen oder Bearbeiten einer Leinwand einen Eckpunkt aus einem
  Polygon löschen
* x : beim Bearbeiten einer Leinwand ein ausgewähltes Objekt löschen
* Ziehen mit links : in der definierten Form auf die Leinwand zeichnen
* Ziehen mit rechts : in der definierten Form auf die Leinwand zeichnen
