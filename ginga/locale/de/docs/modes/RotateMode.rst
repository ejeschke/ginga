Der Rotationsmodus aktiviert Tastenbelegungen, mit denen sich die Achsen
eines Bildes in einem Ginga-Bildbetrachter spiegeln oder vertauschen oder das
Bild drehen lässt.

Modus betreten durch
--------------------
* Leertaste, dann „r“

Modus verlassen durch
---------------------
* Esc

Standardtastenbelegung im Modus
-------------------------------
* „[“ : (umschalten) das Bild in der X-Achse spiegeln
* „{“ : die X-Achse wiederherstellen
* „]“ : (umschalten) das Bild in der Y-Achse spiegeln
* „}“ : die Y-Achse wiederherstellen
* Backslash : (umschalten) die X- und Y-Achse vertauschen
* „|“ : die vertauschten Achsen wieder normal herstellen
* R : Drehung auf 0 Grad zurücksetzen (Spiegelungen/Vertauschungen bleiben)
* t : Spiegelungen oder Vertauschungen zurücksetzen
* Punkt : Bild schrittweise um +90 Grad drehen
* Komma : Bild schrittweise um -90 Grad drehen
* o : das Bild so ausrichten, dass Norden oben und Osten links liegt
* O : das Bild so ausrichten, dass Norden oben und Osten rechts liegt
* Ziehen mit links : das Bild interaktiv drehen
* Rechtsklick : die Drehung auf 0 Grad zurücksetzen (wie R)
* Drehgeste : das Bild interaktiv drehen
