Der Cuts-Modus aktiviert Tastenbelegungen, mit denen sich der untere und
obere Schnittwert eines Bildes in einem Ginga-Bildbetrachter anpassen lassen.

Modus betreten durch
--------------------
* Leertaste, dann „s“

Modus verlassen durch
---------------------
* Esc

Standardtastenbelegung im Modus
-------------------------------
* l : unteren Schnittwert auf den Wert des Pixels unter dem Cursor setzen
* k : oberen Schnittwert auf den Wert des Pixels unter dem Cursor setzen
* S : unteren und oberen Schnittwert auf die Min-/Max-Werte im Bild setzen
* A : unteren und oberen Schnittwert auf 0, 255 setzen;
  meist für normale RGB-Bilder nützlich
* a : automatische Schnittwerte mit dem aktuell gewählten Algorithmus und
  dessen Parametern durchführen
* b, Pfeil hoch : vorherigen Auto-Cuts-Algorithmus der Liste wählen
* n, Pfeil runter : nächsten Auto-Cuts-Algorithmus der Liste wählen
* Doppelpunkt : Auto-Cuts für neue Bilder in diesem Betrachter ein-/ausschalten
* Semikolon : Auto-Cuts für neue Bilder auf „override“ setzen
* Scrollrad : Kontrast durch Stauchen oder Strecken der Werte anpassen;
  eine Richtung staucht, die andere streckt
* Strg + Scrollen : Mikrokontrast durch Stauchen/Strecken anpassen;
  wie Scrollen, aber mit geringerem Ausmaß
* Umschalt + Scrollen : aktuellen Auto-Cuts-Algorithmus wechseln
* Ziehen mit links : Werte durch Cursorbewegung anpassen;
  links/rechts passt den oberen Wert an, hoch/runter den unteren
* Umschalt + Ziehen mit links : unteren Wert durch Cursorbewegung anpassen;
  links/rechts passt den unteren Wert an
* Strg + Ziehen mit links : oberen Wert durch Cursorbewegung anpassen;
  links/rechts passt den oberen Wert an
* Rechtsklick : automatische Werte durchführen (wie „a“)
* Pinch-Geste : Abstand zwischen unterem und oberem Schnittwert vergrößern
  oder verkleinern (ähnlich wie das Scrollrad)
* Strg + Verschiebegeste : oberen Schnittwert nach oben oder unten ändern
* Umschalt + Verschiebegeste : unteren Schnittwert nach oben oder unten ändern
