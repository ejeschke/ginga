このプラグインは、リファレンスビューアのコマンドラインインターフェースを提供
します。

.. note:: このコマンドラインは、プラグイン UI の*内部*で使うためのものです。
          *リモート*のコマンドラインインターフェースをお探しの場合は、``RC``
          プラグインを参照してください。

**プラグインの種類: グローバル**

``Command`` はグローバルプラグインです。開けるインスタンスは 1 つだけです。

**使い方**

コマンドとパラメータの一覧を取得::

        g> help

シェルコマンドを実行::

        g> !cmd arg arg ...

**注意**

特に強力なツールとして、``reload_local`` および ``reload_global`` コマンドを
使って、開発中のプラグインを再読み込みできます。これにより、リファレンス
ビューアを再起動して手間をかけてデータなどを読み込み直す必要がなくなります。
単にプラグインを閉じ、適切な「reload」コマンド（ヘルプを参照!）を実行してから、
プラグインを再び起動します。

.. note:: プラグイン自体*以外*のモジュールを変更した場合、それらはこれらの
          コマンドでは再読み込みされません。
