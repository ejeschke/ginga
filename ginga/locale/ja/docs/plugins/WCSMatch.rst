``WCSMatch`` は Ginga 画像ビューア用のグローバルプラグインで、画像の世界座標系
（WCS）を使って、スケールや向きの異なる画像を表示目的で大まかに位置合わせでき
ます。

**プラグインの種類: グローバル**

``WCSMatch`` はグローバルプラグインです。開けるインスタンスは 1 つだけです。

**使い方**

使用するには、プラグインを起動し、プラグイン GUI の「参照チャンネル」という
ドロップダウンメニューからチャンネルを選択します。そのチャンネルに含まれる
画像が、他のチャンネルの画像を同期させるための基準として使われます。

チャンネルは表示（パン、スケール（ズーム）、変換（反転）、回転）で同期されます。
「パンを合わせる」、「スケールを合わせる」、「変換を合わせる」、「回転を合わせる」
のチェックボックスをオンまたはオフにして、チャンネル間で同期する属性を制御でき
ます。

同期を完全に「解除」するには、「参照チャンネル」ドロップダウンメニューから
「None」を選択するだけです。

現在のところ、プラグインの影響を受けるチャンネルを制限する方法はありません。
