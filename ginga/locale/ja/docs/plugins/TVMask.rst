
ファイルからのマスク（非対話モード）を画像上に表示します。

**プラグインの種類: ローカル**

``TVMask`` はローカルプラグインで、チャンネルに関連付けられています。
チャンネルごとに 1 つのインスタンスを開けます。

**使い方**

このプラグインは、FITS ファイルを読み込むことで、マスクの非対話的な表示を可能に
します。ゼロ以外の値はマスクされたデータと見なされます。

異なるマスクを表示するには（例: 上に示すように、一部を緑、一部を桃色でマスク）:

1. 緑をドロップダウンメニューから選びます。または、希望のアルファ値を入力します。
2. 「マスクを読み込み」ボタンを使って、該当する FITS ファイルを読み込みます。
3. (1) を繰り返しますが、今度は桃色をドロップダウンメニューから選びます。
4. (2) を繰り返しますが、別の FITS ファイルを選びます。
5. 3 つ目のマスクも桃色で表示するには、ドロップダウンメニューを変えずに (4) を
   繰り返します。

表のリストからエントリ（または複数のエントリ）を選択すると、画像上のマスクが
ハイライトされます。ハイライトには事前定義された色とアルファが使われます（下で
カスタマイズ可能）。

このプラグインがアクティブなときに画像上に長方形を描くことで、ある領域内の
すべてのマスクを画像上と表のリストの両方でハイライトすることもできます。

「隠す」ボタンを押すとマスクが隠れますが、プラグインのメモリは消去されません。
つまり、「表示」を押すと、同じマスクが同じ画像上に再び現れます。ただし、「忘れる」
を押すと、マスクは表示とメモリの両方から消去されます。つまり、マスクを再作成する
にはファイルを再読み込みする必要があります。

同じマスクを別の色やアルファで再描画するには、「忘れる」を押し、必要に応じて上記の
手順を繰り返します。

指向／寸法が大きく異なる画像が同じチャンネルに表示されている場合、ある画像に属する
が別の画像の外に出るマスクは、後者には表示されません。

このプラグインが読めるマスクを作成するには、``Drawing`` プラグインの結果を使う
ことができます（描画後に「マスクを作成」を押し、``SaveImage`` を使ってマスクを
保存）。ほかに ``astropy.io.fits`` などを使って手動で FITS ファイルを作成すること
もできます。

``TVMark`` と併用すると、Ginga で点源とマスク領域の両方を重ねて表示できます。

``~/.ginga/plugin_TVMask.cfg`` を使ってカスタマイズできます。``~`` はあなたの
HOME ディレクトリです:

.. code-block:: Python

  #
  # TVMask plugin preferences file
  #
  # Place this in file under ~/.ginga with the name "plugin_TVMask.cfg"

  # Mask color -- Any color name accepted by Ginga
  maskcolor = 'green'

  # Mask alpha (transparency) -- 0=transparent, 1=opaque
  maskalpha = 0.5

  # Highlighted mask color and alpha
  hlcolor = 'white'
  hlalpha = 1.0
