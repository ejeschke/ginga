``Zoom`` プラグインは、関連するチャンネル画像内のカーソル位置の下を中心とした
切り出し領域の拡大画像を表示します。カーソルを画像上で動かすと、ズーム画像が
更新され、ピクセルの詳細な検査や、他のプラグイン操作と組み合わせた精密な制御が
可能になります。

**プラグインの種類: グローバル**

``Zoom`` はグローバルプラグインです。開けるインスタンスは 1 つだけです。

**使い方**

ズームウィンドウの拡大率は、「ズーム量」スライダーを調整して変更できます。

2 つの動作モードが可能です -- 絶対ズームと相対ズーム:

* 絶対モードでは、ズーム量が切り出しに表示されるズームレベルを正確に制御します。
  例えば、チャンネル画像が 10 倍に拡大されていても、ズーム量が 3 倍に設定されて
  いれば、ズーム画像は 3 倍の画像のみを表示します。

* 相対モードでは、ズーム量の設定はチャンネル画像のズーム設定に対する相対値として
  解釈されます。ズーム量が 3 倍に設定され、チャンネル画像が 10 倍にズームされて
  いる場合、表示されるズーム画像は 13 倍（10 倍 + 3 倍）になります。ズーム量の
  設定は 1 未満にもできるので、チャンネル画像が 3 倍ズームのときに 1/3 倍の設定に
  すると、1 倍のズーム画像が得られます。

「更新間隔」の設定は、``Zoom`` プラグインがカーソルの動きに応じてズーム画像を
更新する速さを制御します。値はミリ秒で指定します。

.. tip:: 通常、小さな更新間隔を設定するとズーム画像の全体的な応答性が*向上*し、
         既定値の 20 は妥当な値です。ズーム画像がぎこちなく見えたり、チャンネル
         画像ウィンドウ内のマウスの動きと同期していないように見える場合は、値を
         いろいろ試してみてください。

「既定値」ボタンは、コントロールの既定設定を復元します。

``~/.ginga/plugin_Zoom.cfg`` を使ってカスタマイズできます。``~`` はあなたの
HOME ディレクトリです:

.. code-block:: Python

  #
  # Zoom plugin preferences file
  #
  # Place this in file under ~/.ginga with the name "plugin_Zoom.cfg"

  # default zoom level
  zoom_amount = 3

  # refresh interval (sec)
  # NOTE: usually a small delay speeds things up
  refresh_interval = 0.02
