``AutoLoad`` は、フォルダを監視して新しいファイルを検出し、出現したときに自動的
にチャンネルへ読み込むシンプルなプラグインです。

**プラグインの種類: ローカル**

``AutoLoad`` はローカルプラグインで、チャンネルに関連付けられています。
チャンネルごとに 1 つのインスタンスを開けます。

.. note:: このプラグインを使うには、Python の「watchdog」パッケージを
          インストールする必要があります。

**使い方**

* 監視するフォルダを設定するには、「監視フォルダ」フィールドにフォルダ
  （ディレクトリ）のパスを入力し、ENTER を押すか「設定」をクリックします。
* このフォルダに追加されるファイルを区別する必要がある場合は、「正規表現」
  ボックスに Python の正規表現を入力して「設定」をクリックできます。名前が
  パターンに一致するファイルのみが対象になります。正規表現はファイル名のみに
  適用され、フォルダパスの一部には適用されないことに注意してください。
* 自動読み込みを一時停止したい場合は、「自動読み込みを一時停止」のボックスに
  チェックを入れると、すべての自動読み込みが停止します。その後ボックスの
  チェックを外しても、その間に到着したファイルは読み込まれないことに注意して
  ください。

.. note:: ネットワークドライブ上にあるフォルダの監視は、機能する場合と機能
          しない場合があります。
