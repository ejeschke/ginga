``Header`` プラグインは、画像に関連付けられたメタデータの一覧を提供します。

**プラグインの種類: グローバル**

``Header`` はグローバルプラグインです。開けるインスタンスは 1 つだけです。

**使い方**

``Header`` プラグインは画像の FITS キーワードメタデータを表示します。
最初はプライマリ HDU のメタデータのみが表示されます。ただし、``MultiDim``
プラグインと組み合わせると、他の HDU のメタデータも表示されます。詳しくは
``MultiDim`` を参照してください。

UI の左下にある「並べ替え可能」チェックボックスがオンになっている場合、列の
見出しをクリックするとその列の値でテーブルが並べ替えられ、特定のキーワードを
素早く見つけるのに役立ちます。

「プライマリヘッダを含める」チェックボックスは、プライマリ HDU のキーワードを
含めるかどうかを切り替えます。画像がプライマリヘッダを保存しないオプションで
作成された場合、このオプションは無効になることがあります。
