チャンネルビューアの画像を PNG または JPEG 画像として取得します。

**使い方**

1. 「種類」コンボボックスからスナップの RGB グラフィックタイプを選択します。
2. 取得したい状態にチャンネル画像がなったら「スナップ」を押します。

RGB 画像のコピーが ``ScreenShot`` ビューアに読み込まれます。
通常の Ginga ビューアと同じように ``ScreenShot`` ビューア内をパン・ズームして
詳細を調べられます（例: JPEG と PNG 形式の拡大した違いを見る）。

3. 目的の画像になるまで (1) と (2) を繰り返します。
4. 「フォルダ」テキストボックスに新しいファイルの有効なパスを入力します。
5. 「名前」テキストボックスに新しいファイルの有効な名前を入力します。
   ファイル拡張子を追加する必要はありません。必要に応じて追加されます。
6. 「保存」ボタンを押します。指定した場所にファイルが保存されます。

**注意**

* PNG は重ねたグラフィックのアーティファクトが少ないですが、ファイルは JPEG
  より大きくなります。
* 「中央寄せ」ボタンはスナップ画像を中央に配置し、「フィット」はウィンドウに
  合わせてズームを設定し、「消去」は画像を消去します。「全体」を押すと 100 %
  ピクセル（1:1 スケール）にズームします。
* 「画面サイズ」チェックボックス（既定でオン）は、画像をチャンネルビューア
  ウィンドウとまったく同じサイズで保存します。別のサイズで保存するには、この
  ボックスのチェックを外し、「幅」と「高さ」ボックスでサイズを設定します。
* 「縦横比を固定」機能は「画面サイズ」がオフのときのみ機能します。有効な場合、
  幅または高さを変更すると、「縦横比」ボックスに表示された比率を保つために
  もう一方のパラメータが変更されます。
