``SAMP`` プラグインは、Ginga リファレンスビューア用の SAMP インターフェースを
実装します。

.. note:: このプラグインを実行するには、``samp`` モジュールを含む ``astropy``
          をインストールする必要があります。

**プラグインの種類: グローバル**

``SAMP`` はグローバルプラグインです。開けるインスタンスは 1 つだけです。

**使い方**

Ginga には、SAMP（Simple Applications Messaging Protocol）サポートを有効にする
ためのプラグインが含まれています。SAMP サポートにより、Ginga は他の天文デスク
トップアプリケーションから制御でき、相互運用できます。

``SAMP`` モジュールは既定では起動しません。Ginga の起動時に起動するには、コマンド
ラインオプションを指定します::

        --modules=SAMP

そうでなければ、「プラグイン」メニューの「SAMP ハブを起動」から起動します。

現在、SAMP サポートは ``image.load.fits`` メッセージに限られており、Ginga は
これらのメッセージのいずれかを受信すると FITS ファイルを読み込みます。

Ginga の ``SAMP`` プラグインは ``astropy.samp`` モジュールを使用するため、
プラグインを使うには ``astropy`` がインストールされている必要があります。
既定では、Ginga の ``SAMP`` プラグインは、実行中の SAMP ハブが見つからない場合、
ハブの起動を試みます。
