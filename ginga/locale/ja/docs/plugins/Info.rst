
``Info`` プラグインは、フォーカスされているチャンネル画像に関する、よく役立つ
メタデータのペインを提供します。一般的な情報には、いくつかのメタデータヘッダ値、
座標、画像の寸法、最小値・最大値などが含まれます。カーソルを画像上で動かすと、
X、Y、Value、RA、DEC の値が更新され、カーソルの下の値を反映します。

**プラグインの種類: グローバル**

``Info`` はグローバルプラグインです。開けるインスタンスは 1 つだけです。

**使い方**

``Info`` インターフェースの下部には、色分布とカットレベルのコントロールがあります。
カットレベルのボックスの上にあるセレクタで、画像の値をカラーマップにマッピングする
いくつかの分布アルゴリズムから選べます。選択肢は「linear」「log」「power」「sqrt」
「squared」「asinh」「sinh」「histeq」（ヒストグラム平坦化）です。

その下に、低・高カットレベルが表示され、調整できます。「Auto Levels」ボタンを
押すと、現在の自動カットレベルアルゴリズムとチャンネル設定で定義されたパラメータに
基づいて、カットレベルが再計算されます。

「Auto Levels」ボタンの下には、現在アクティブなチャンネルについて「Cut New」
「Zoom New」「Center New」の設定の状態が表示されます。これらは、チャンネルに
追加される新しい画像が、自動カットレベル、ウィンドウへの合わせ、画像中央への
パンによってどのように影響されるかを示します。

「Follow New」チェックボックスは、チャンネルに追加された新しい画像をビューアが自動
的に表示するかどうかを制御します。「Raise New」チェックボックスは、新しい画像が
追加されたときに画像ビューアウィンドウを前面に出すかどうかを制御します。これらの
2 つのコントロールは、例えば外部プログラムがビューアに画像を追加していて、特定の
画像を調べる作業の中断を防ぎたい場合に便利です。

グローバルプラグインとして、``Info`` は新しいチャンネルへのフォーカスの変化に応じて、
新しいチャンネルのメタデータを表示します。通常、ユーザーインターフェースの
「Synopsis」タブの下に現れます。

このプラグインは通常、閉じられるようには構成されていませんが、設定ファイルで
「closeable」設定を True にすることで、そうできます。すると UI の下部に「閉じる」
ボタンと「ヘルプ」ボタンが追加されます。
