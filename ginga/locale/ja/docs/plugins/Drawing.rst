キャンバスの図形（重ねて表示するグラフィック）を描画するためのプラグインです。

**プラグインの種類: ローカル**

``Drawing`` はローカルプラグインで、チャンネルに関連付けられています。
シングルトンではないため、チャンネルごとに複数のインスタンスを開けます。

**使い方**

このプラグインを使って、画像表示上にさまざまな図形を描けます。「描画」モードの
とき、ドロップダウンメニューから図形を選び、（必要に応じて）図形のパラメータを
調整し、マウスの左ボタンで画像に描画します。ピクセル空間または WCS 空間で描画
できます。

既存の図形を移動または編集するには、プラグインをそれぞれ「編集」または「移動」
モードにします。

描いた図形をマスク画像として保存するには、「マスクを作成」ボタンをクリックします。
すると Ginga に新しいマスク画像が作成されます。次に ``SaveImage`` プラグインを
使って、単一拡張の FITS として保存します。マスクは表示中の画像のサイズになる
ことに注意してください。したがって、異なる画像寸法のマスクを作成するには、この
手順を複数回繰り返す必要があります。

キャンバスに描いた図形は、astropy-regions 形式（DS9 の領域と互換）で読み込みや
保存ができます。使うには astropy-regions パッケージがインストールされている
必要があります。キャンバスに座標を「data」（ピクセル）または「wcs」としてオブ
ジェクトを描くだけです。すべての Ginga キャンバスオブジェクトが領域の図形に
変換できるわけではなく、一部の属性は保存されない、無視される、または他の
ソフトウェアで領域の図形を読み込む際にエラーの原因になることがある点に注意して
ください。
