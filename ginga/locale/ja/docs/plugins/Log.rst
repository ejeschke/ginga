リファレンスビューアのログ出力を表示します。

**プラグインの種類: グローバル**

``Log`` はグローバルプラグインです。開けるインスタンスは 1 つだけです。

**使い方**

``Log`` プラグインは、ロガーの現在の出力を表示する大きなスクロール可能な
テキストウィジェットを含む UI を構築します。最新の出力は下部に表示されます。
これは問題のトラブルシューティングに役立ちます。

コントロールは 4 つあります:

* 左下のコンボボックスで、希望するログレベルを選択できます。冗長度の順に、
  4 つのレベルは「debug」、「info」、「warn」、「error」です。
* 右下の数値のボックスで、表示バッファに保持する入力行数を設定できます
  （例: 最後の 1000 行だけ保持する）。
* 「自動スクロール」チェックボックスをオンにすると、新しいログメッセージが
  追加されるたびに大きなテキストウィジェットが末尾までスクロールします。
  古いメッセージをじっくり見たい場合はオフにしてください。
* 「消去」ボタンはテキストウィジェットを消去し、新しいログだけが表示される
  ようにします。
