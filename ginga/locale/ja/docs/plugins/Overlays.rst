読み込んだ画像の露出不足・露出過多を表す色オーバーレイを生成するプラグインです。

**プラグインの種類: ローカル**

``Overlays`` はローカルプラグインで、チャンネルに関連付けられています。
チャンネルごとに 1 つのインスタンスを開けます。

**使い方**

下限や上限の色をドロップダウンメニューから選びます（それぞれ「下限の色」と
「上限の色」）。下限値と上限値を上限フィールドに指定します（それぞれ「下限」と
「上限」）。オーバーレイの不透明度を「不透明度」フィールドに 0 から 1 の値で
設定します。最後に「やり直す」ボタンを押します。

色オーバーレイは、下限より下の領域を下限の色で、上限より上の領域を上限の色で
表示します。いずれかの限界を省略すると（フィールドを空欄のままにすると）、
その色はオーバーレイに表示されません。

チャンネルに新しい画像が選択されると、オーバーレイ画像は現在のパラメータに
基づいて新しいデータで再計算されます。
