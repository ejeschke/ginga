
Ginga リファレンスビューアのダウンロード GUI。

**プラグインの種類: グローバル**

``Download`` はグローバルプラグインです。開けるインスタンスは 1 つだけです。

**使い方**

このプラグインを開いて、URI ダウンロードの進行状況を監視します。「Plugins」または
「Operations」メニューを使って起動し、「Util」カテゴリの下から「Downloads」
プラグインを選択します。

ダウンロードを開始したい場合は、URI をチャンネル画像ビューアまたは ``Thumbs``
ペインにドラッグするだけです。

各エントリの「消去」ボタンをクリックすることで、いつでもダウンロードに関する情報を
削除できます。下部の「すべて消去」ボタンをクリックすると、すべてのダウンロードの
エントリを消去できます。

現在、進行中のダウンロードをキャンセルすることはできません。

**設定**

``auto_clear_download`` オプションを `True` に設定すると、ダウンロードが完了した
ときに、ダウンロードエントリがペインから自動的に削除されます。ダウンロードされた
ファイルは削除されません。

ダウンロードフォルダは、~/.ginga/general.cfg の「download_folder」設定に値を割り
当てることで、ユーザーが定義できます。割り当てられていない場合は、既定で、
プラットフォーム固有の既定の一時ディレクトリ内のフォルダになります（Python の
「tempfile」モジュールが示すとおり）。
