
``Histogram`` は、画像内に描いた領域、または画像全体のヒストグラムをプロットし
ます。

**プラグインの種類: ローカル**

``Histogram`` はローカルプラグインで、チャンネルに関連付けられています。
シングルトンではないため、チャンネルごとに複数のインスタンスを開けます。

**使い方**

クリックしてドラッグし、ヒストグラムの計算に使う領域を画像内に定義します。画像
全体のヒストグラムを取るには、UI の「画像全体」とラベル付けされたボタンをクリック
します。

.. note:: 画像のサイズによっては、完全なヒストグラムの計算に時間がかかることが
          あります。

チャンネルに新しい画像が選択されると、現在のパラメータに基づいて新しいデータで
ヒストグラムプロットが再計算されます。

ヒストグラムプラグインの設定ファイルで無効にされていない限り、ボックスに対する
簡単な統計の行が計算され、プロットの下の行に表示されます。

**UI コントロール**

UI 下部の 3 つのラジオボタンは、クリック／ドラッグ操作の効果を制御するために使い
ます:

* 領域を別の場所にドラッグするには「移動」を選びます
* 新しい領域を描くには「描画」を選びます
* 領域を編集するには「編集」を選びます

ヒストグラムの対数プロットを作るには、「対数ヒストグラム」チェックボックスを
チェックします。カット値の範囲内ではなく画像の値の全範囲でプロットするには、
「カットでプロット」チェックボックスのチェックを外します。

「NumBins」パラメータは、ヒストグラムの計算に使うビンの数を決めます。ボックスに
数値を入力し、「Enter」を押して既定値を変更します。

**カットレベルの便利なコントロール**

ヒストグラムはカットレベルの設定に役立つフィードバックなので、UI には、画像内の
低・高カットレベルを設定するコントロールや、チャンネル設定の自動カットレベル設定に
従って自動カットレベルを実行するコントロールが用意されています。

ヒストグラムプロット内をクリックしてカットレベルを設定できます:

* 左クリック: 低カットを設定
* 中クリック: リセット（自動カットレベル）
* 右クリック: 高カットを設定

さらに、プロット内でホイールをスクロールすることで、低カットと高カットの間の幅
（すなわちヒストグラムプロット曲線の「幅」）を動的に調整できます。これは画像内の
コントラストを増減させる効果があります。ホイールクリックごとに変更される量は、
プラグイン構成ファイルの設定 ``scroll_pct`` で設定されます。既定は 10 % です。

**ユーザー構成**

``~/.ginga/plugin_Histogram.cfg`` を使ってカスタマイズできます。``~`` はあなたの
HOME ディレクトリです:

.. code-block:: Python

  #
  # Histogram plugin preferences file
  #
  # Place this in file under ~/.ginga with the name "plugin_Histogram.cfg"

  # Switch to "move" mode after selection
  draw_then_move = True

  # Number of bins for histogram
  num_bins = 2048

  # Histogram color
  hist_color = 'aquamarine'

  # Calculate extra statistics on box
  show_stats = True

  # Controls formatting (width) of statistics numbers
  maxdigits = 7

  # percentage to adjust cuts gap when scrolling in histogram
  scroll_pct = 0.10
