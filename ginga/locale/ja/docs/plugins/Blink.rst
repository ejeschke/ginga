``Blink`` は、ユーザーが選んだ速度でチャンネルに表示された画像を切り替えます。
あるいは、メインのワークスペース内のチャンネル間を切り替えることもできます。
どちらの場合も、主な目的は、まばたきのように短い時間内で画像を（チャンネル内で、
またはチャンネル間で）視覚的に比較・対照することです。

**プラグインの種類: ローカルまたはグローバル**

``Blink`` はローカルプラグインとして呼び出すとチャンネル内の画像を順に切り替え、
グローバルプラグインとして呼び出すとチャンネルを順に切り替えます。

ローカルプラグインは「操作」ボタンから、グローバルプラグインは「プラグイン」
メニューから起動します。

**使い方**

画像切り替えの間隔を「間隔」フィールドに秒単位で設定します。次に「点滅を開始」
を押すと時間指定の切り替えが始まり、「点滅を停止」を押すと止まります。

「間隔」の数値を変更して ``Enter`` を押すと、サイクルの実行中にサイクル時間を
動的に変更できます。
