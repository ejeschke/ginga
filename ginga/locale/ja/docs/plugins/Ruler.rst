``Ruler`` は、画像上の距離を測定するために設計されたシンプルなプラグインです。

**プラグインの種類: ローカル**

``Ruler`` はローカルプラグインで、チャンネルに関連付けられています。
シングルトンではないため、チャンネルごとに複数のインスタンスを開けます。

**使い方**

``Ruler`` は、画像上に描いた 1 本の線で定義される 3 点の WCS マッピングを通じて
球面三角測量を計算することで距離を測定します。既定では距離は天球上の分角で表示
されますが、「単位」コントロールを使って、代わりに度またはピクセル距離で表示する
ように変更できます。

クリックしてドラッグすると、2 点間に定規が設定されます。描画操作を終えると定規が
設定され、プラグイン UI が更新されて、端点の位置や線の角度など、線に関する詳細が
表示されます。角度の単位は、隣接するドロップダウンボックスで度とラジアンの間で
切り替えられます。

古い定規を消して新しい定規を作るには、もう一度クリックしてドラッグします。別の線
を描くと、最初の線を置き換えます。プラグインを閉じると、グラフィックオーバーレイ
は削除されます。「固定される定規」が欲しい場合は、``Drawing`` プラグインを使い
（描画タイプとして「Ruler」を選択）ます。

**編集**

既存の定規を編集するには、プラグイン UI の「編集」とラベル付けされたラジオ
ボタンをクリックします。定規がすぐに選択されない場合は、2 点を結ぶ対角線を
クリックします。これで定規の周りにバウンディングボックスが作られ、そのコントロール
点が表示されるはずです。バウンディングボックス内をドラッグすると定規を移動でき、
端点をクリックしてドラッグすると定規を編集できます。これらのコントロール点を使って
定規を拡大縮小したり回転させたりすることもできます。

**UI**

距離に表示される単位は、UI のドロップダウンボックスから選択できます。「arcmin」、
「degrees」、「pixels」から選べます。最初の 2 つは、画像に有効で機能する WCS が
必要です。

端点の値は UI に表示されますが、「端点を表示」チェックボックスを切り替えると、
定規のグラフィックにも追加で表示できます。「垂線を表示」ボックスを切り替えると、
垂線が表示されます。

**ボタン**

「始点にパン」ボタンはメイン画像を、描いた線の起点にパンし、「終点にパン」は終点に
パンします。「中心にパン」はパン位置を線の中心点に設定します。これらのボタンは、
画像の接近したズームイン作業に役立つことがあります。「消去」は画像から定規を消去
します。

**ヒント**
「Zoom」プラグインを開くと、カーソル領域の詳細を正確に見られます。「Pick」
プラグインも Ruler と組み合わせて使うことで、定規のいずれかの端を合わせる際に、
オブジェクトの中心点を特定できます。
