
構成モノクロ画像から RGB 画像を合成するためのプラグインです。

**プラグインの種類: ローカル**

``Compose`` はローカルプラグインで、チャンネルに関連付けられています。
チャンネルごとに 1 つのインスタンスを開けます。

**使い方**

``Compose`` プラグインを「Operation->RGB」（下）または「Plugins->RGB」（上）
メニューから起動します。タブは右側のビューアの「Dialogs」タブの下に
「IMAGE:Compose」として表示されるはずです。

1. 作りたい合成の種類を「Compose Type」ドロップダウンから選びます: 3 枚の
   モノクロ画像を 1 枚のカラー画像に合成する場合は「RGB」、一連の画像を層ごとに
   異なるアルファ値でレイヤーとして合成する場合は「Alpha」。
2. 「新しい画像」を押して、新しい画像の合成を開始します。

***RGB 合成の場合***

1. R、G、B の各プレーンを構成する 3 枚の画像を「Preview」ウィンドウにドラッグ
   します -- R（赤）、G（緑）、B（青）の順にドラッグします。あるいは、画像を 1 枚
   ずつチャンネルビューアに読み込み、それぞれの後に「チャンネルから挿入」を押しても
   よいです（同様に R、G、B の順で行います）。

プラグイン GUI では、R、G、B の画像がプラグインの「Layers」エリアに 3 つの
スライダーコントロールとして表示され、プレビューには、スライダーを設定した状態で
合成画像がどのように見えるかの低解像度版が表示されるはずです。

.. figure:: figures/compose-rgb.png
   :width: 800px
   :align: center
   :alt: RGB 画像の合成

   RGB 画像の合成。

2. ``Compose`` プラグインのスライダーを使って各レイヤーのアルファレベルをいろいろ
   試してください。スライダーを調整すると、プレビュー画像が更新されるはずです。
3. 気に入ったものが見つかったら、「名前を付けて保存」ボタンでファイルに保存する
   （ファイル拡張子として「jpeg」または「png」を使う）か、「チャンネルに保存」
   ボタンでチャンネルに挿入できます。

***Alpha 合成の場合***

Alpha タイプの合成では、画像はスタックに表示された順序で単純に結合され、レイヤー 0
が最下層で、後続のレイヤーがその上に積み重ねられます。各レイヤーのアルファレベルは、
上で説明したのと同じ方法でスライダーによって調整できます。

.. figure:: figures/compose-alpha.png
   :width: 800px
   :align: center
   :alt: 画像のアルファ合成

   画像のアルファ合成。

1. レイヤーを構成する N 枚の画像を「Preview」ウィンドウにドラッグするか、画像を
   1 枚ずつチャンネルビューアに読み込み、それぞれの後に「チャンネルから挿入」を
   押します（最初の画像がスタックの最下部 -- レイヤー 0 になります）。
2. ``Compose`` プラグインのスライダーを使って各レイヤーのアルファレベルをいろいろ
   試してください。スライダーを調整すると、プレビュー画像が更新されるはずです。
3. 気に入ったものが見つかったら、「名前を付けて保存」ボタンでファイルに保存する
   （ファイル拡張子として「fits」を使う）か、「チャンネルに保存」ボタンで
   チャンネルに挿入できます。

***一般的な注意***

- プレビューウィンドウは単なる ginga ウィジェットなので、通常のすべてのバインディング
  が適用されます。マウスやキーのバインディングでカラーマップやカットレベルなどを
  設定できます。
