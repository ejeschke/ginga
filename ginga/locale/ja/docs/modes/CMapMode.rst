CMap モードは、Ginga 画像ビューアで画像のカラーマップと強度マップを
調整するためのキー割り当てを有効にします。

モードに入る方法
----------------
* スペース、次に「y」

モードを抜ける方法
------------------
* Esc

モードの標準キー割り当て
------------------------
* Y : カラーマップを既定（グレー）にリセット
* r : カラーマップを復元（反転・回転・ストレッチを取り消す）
* I : 現在のカラーマップを反転（回転やストレッチは保持）
* 上矢印 : リストの前のカラーマップを設定（回転やストレッチは保持）
* 下矢印 : リストの次のカラーマップを設定（回転やストレッチは保持）
* c : 見えるカラーマップを画像に重ねて表示
* i : 既定の強度マップ「ramp」を復元
* j、左矢印 : リストの前の強度マップを設定
* k、右矢印 : リストの次の強度マップを設定
* スクロール : リストからカラーマップを選択
* Ctrl + スクロール : リストから強度マップを選択
* パンジェスチャ : リストからカラーマップを選択（ヒント: 上下矢印キーで
  選択を確定）
* 左ドラッグ : 現在のカラーマップを回転
* 右クリック : カラーマップを復元（「r」と同じ）
