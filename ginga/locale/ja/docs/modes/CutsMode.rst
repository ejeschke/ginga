Cuts モードは、Ginga 画像ビューアで画像の下限・上限カットレベルを
調整するためのキー割り当てを有効にします。

モードに入る方法
----------------
* スペース、次に「s」

モードを抜ける方法
------------------
* Esc

モードの標準キー割り当て
------------------------
* l : 下限カットレベルをカーソル下のピクセル値に設定
* k : 上限カットレベルをカーソル下のピクセル値に設定
* S : 下限・上限カットレベルを画像内の最小／最大値に設定
* A : 下限・上限カットレベルを 0、255 に設定;
  主に標準的な RGB 画像に便利
* a : 現在選択中の自動カットアルゴリズムとパラメータで自動カットを実行
* b、上矢印 : リストの前の自動カットアルゴリズムを選択
* n、下矢印 : リストの次の自動カットアルゴリズムを選択
* コロン : このビューアで新しい画像の自動カットのオン／オフを切り替え
* セミコロン : このビューアで新しい画像の自動カットを「override」に設定
* スクロールホイール : レベルを狭めたり広げたりしてコントラストを調整;
  一方向は狭め、もう一方は広げる
* Ctrl + スクロール : レベルを狭め／広げて微細なコントラストを調整;
  スクロールと同様だが変化量は小さい
* Shift + スクロール : 現在の自動カットアルゴリズムを変更
* 左ドラッグ : カーソル移動でレベルを調整;
  左右で上限、上下で下限を調整
* Shift + 左ドラッグ : カーソル移動で下限を調整; 左右で下限を調整
* Ctrl + 左ドラッグ : カーソル移動で上限を調整; 左右で上限を調整
* 右クリック : 自動レベルを実行（「a」と同じ）
* ピンチジェスチャ : 下限と上限カットレベルの間隔を広げる／狭める
  （スクロールホイールと同様）
* Ctrl + パンジェスチャ : 上限カットレベルを上下に変更
* Shift + パンジェスチャ : 下限カットレベルを上下に変更
