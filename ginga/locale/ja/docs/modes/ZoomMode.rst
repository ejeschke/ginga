Zoom モードは、Ginga 画像ビューアでパン位置（中央ピクセル）とズームレベル
（スケール）を設定するためのキー割り当てを有効にします。

スクロールとパンの操作方法が Pan モードとは異なります。

モードに入る方法
----------------
* スペース、次に「w」

モードを抜ける方法
------------------
* Esc

モードの標準キー割り当て
------------------------
* 中ドラッグ : 画像内を自由にパン
* 左クリック : カーソル下にパンして拡大
* 右クリック : 縮小
* パンジェスチャ : 画像を拡大／縮小
* Shift + パンジェスチャ : 原点を設定して画像を拡大／縮小
