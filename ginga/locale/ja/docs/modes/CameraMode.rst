カメラモードは、OpenGL バックエンドで動作している Ginga 画像ビューアで
カメラを操作するためのキー割り当てを有効にします。

.. note:: カメラモードは、ビューアが OpenGL バックエンドを使用していない
          場合は機能しません。

モードに入る方法
----------------
* スペース、次に「c」

モードを抜ける方法
------------------
* Esc

モードの標準キー割り当て
------------------------
* s : 後で復元できるよう現在のカメラ設定を保存
* r : 保存したカメラ設定に戻す
* 3 : カメラビューを 3D モードに切り替え
* スクロール : カメラを追従（ドリーイン／アウト）
* 左ドラッグ : カメラでビューを周回
* 右ドラッグ : カメラでビューをパン
