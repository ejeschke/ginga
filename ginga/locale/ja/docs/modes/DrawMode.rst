描画モードは、描画を容易にするキー割り当てを有効にします。

モードに入る方法
----------------
* スペース、次に「b」

モードを抜ける方法
------------------
* Esc

モードの標準キー割り当て
------------------------
* v : キャンバスの描画中または編集中に多角形オブジェクトへ頂点を追加
* z : キャンバスの描画中または編集中に多角形オブジェクトから頂点を削除
* x : キャンバスの編集中に選択したオブジェクトを削除
* 左ドラッグ : 定義された形状でキャンバスに描画
* 右ドラッグ : 定義された形状でキャンバスに描画
