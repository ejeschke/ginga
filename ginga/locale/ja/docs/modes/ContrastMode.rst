コントラストモードは、Ginga 画像ビューアで画像のコントラストを調整する
ためのキー割り当てを有効にします。

モードに入る方法
----------------
* スペース、次に「t」

モードを抜ける方法
------------------
* Esc

モードの標準キー割り当て
------------------------
* T : コントラストを既定値に戻す
* 左ドラッグ : コントラストを調整
  * カラーマップを対話的にシフト／ストレッチ（コントラストとバイアス）。
  * 左右でシフト、上下でストレッチを操作。
  * コントラストが決まったらボタンを離す。
* 右クリック : コントラストを既定値に戻す
* スクロールホイール : コントラストを変更（Ctrl でより細かく調整）
* Shift + スクロールホイール : 明るさを変更（Ctrl でより細かく調整）
* Ctrl + パンジェスチャ : コントラストを変更
* Shift + パンジェスチャ : 明るさを変更
