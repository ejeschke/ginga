回転モードは、Ginga 画像ビューアで画像の軸を反転・入れ替えたり、画像を
回転したりするためのキー割り当てを有効にします。

モードに入る方法
----------------
* スペース、次に「r」

モードを抜ける方法
------------------
* Esc

モードの標準キー割り当て
------------------------
* 「[」: （切り替え）画像を X 軸で反転
* 「{」: X 軸を元に戻す
* 「]」: （切り替え）画像を Y 軸で反転
* 「}」: Y 軸を元に戻す
* バックスラッシュ : （切り替え）X 軸と Y 軸を入れ替え
* 「|」: 入れ替えた軸を通常に戻す
* R : 回転を 0 度にリセット（反転や入れ替えはリセットしない）
* t : 反転や入れ替えをリセット
* ピリオド : 画像を +90 度ずつ回転
* カンマ : 画像を -90 度ずつ回転
* o : 北が上、東が左になるよう画像を向ける
* O : 北が上、東が右になるよう画像を向ける
* 左ドラッグ : 画像を対話的に回転
* 右クリック : 回転を 0 度にリセット（R と同じ）
* 回転ジェスチャ : 画像を対話的に回転
