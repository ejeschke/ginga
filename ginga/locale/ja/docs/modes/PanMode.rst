Pan モードは、Ginga 画像ビューアでパン位置（中央ピクセルの下の位置）と
ズームレベル（スケール）を設定するためのキー割り当てを有効にします。

モードに入る方法
----------------
* スペース、次に「q」

モードを抜ける方法
------------------
* Esc

モードの標準キー割り当て
------------------------
* プラス、イコール : 1 ズームレベル拡大
* マイナス、アンダースコア : 1 ズームレベル縮小
* 1-9,0 : レベル N にズーム（0 は 10）
* Shift + 1-9,0 : レベル -N にズーム（0 は -10）
* バッククォート : 画像をウィンドウサイズに合わせる
* ダブルクォート : このビューアで新しい画像の自動フィットのオン／オフを切り替え
* シングルクォート : このビューアで新しい画像の自動フィットを「override」に設定
* p : カーソル下の位置にパン
* c : 画像の中心にパン
* z : ズームレベル（スケール）を保存
* r : カーソルにパンして保存したスケールレベルにズーム
* 左／右／上／下矢印 : 小さな割合で左／右／上／下にパン
* Shift + 矢印キー : ごく小さな割合で左／右／上／下にパン
* PageUp（PageDown）: 画面の大きな割合で上（下）にパン
* Home（End）: 画像の上端（下端）方向へパン
* 「?」: このビューアで新しい画像の自動センタリングのオン／オフを切り替え
* 「/」: このビューアで新しい画像の自動センタリングを「override」に設定
* スクロール : 画像をズーム（スケール）
* 左ドラッグ : ビューをパン
* 右ドラッグ : カメラでビューをパン
* パンジェスチャ : ジェスチャに比例してビューをパン
* ピンチジェスチャ : ジェスチャに比例して画像をズーム
