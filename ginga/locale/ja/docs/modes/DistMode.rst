Dist モードは、Ginga 画像ビューアで画像の色分布を調整するための
キー割り当てを有効にします。

これらのアルゴリズムは「トーンカーブ」プロファイルに似ています:「linear」、
「log」、「power」、「sqrt」、「squared」、「asinh」、「sinh」、「histeq」。

モードに入る方法
----------------
* スペース、次に「d」

モードを抜ける方法
------------------
* Esc

モードの標準キー割り当て
------------------------
* D : 色分布アルゴリズムを「linear」にリセット
* b、上矢印 : リストの前の分布アルゴリズムを選択
* n、下矢印 : リストの次の分布アルゴリズムを選択
* スクロールホイール : スクロールで色分布アルゴリズムを選択
* パンジェスチャ : スワイプで色分布アルゴリズムを選択
  （ヒント: 上下矢印キーでアルゴリズムの選択を確定）
