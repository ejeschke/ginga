Plot2D モードは、Matplotlib プロットビューア（ginga.plot.PlotView モジュール）
でパン位置とズームレベル（スケール）を設定するためのキー割り当てを
有効にします。

モードに入る方法
----------------
* スペース、次に「p」

モードを抜ける方法
------------------
* Esc

モードの標準キー割り当て
------------------------
* Shift + 左クリック : パン位置を設定
* 中クリック : パン位置を設定
* スクロール : 拡大／縮小
* Ctrl + スクロール : X 軸のみ拡大／縮小
* Shift + スクロール : Y 軸のみ拡大／縮小（MacOS ではトラックパッドのみ）
* Alt + スクロール（マウス）: Y 軸のみ拡大／縮小（Mac では Option キー）
* Alt + スクロール : カーソル位置で拡大／縮小

キーストローク割り当て
----------------------

ズーム
------
* イコール : 1 ズームレベル拡大
* Ctrl + イコール : X 軸を 1 ズームレベル拡大
* プラス（Shift + イコール）: Y 軸を 1 ズームレベル拡大
* マイナス : 1 ズームレベル縮小
* Ctrl + マイナス : X 軸を 1 ズームレベル縮小
* アンダースコア（Shift + マイナス）: Y 軸を 1 ズームレベル縮小
* 9 : カーソル位置を維持して縮小
* Ctrl + 9 : カーソル位置を維持して X 軸を縮小
* 左括弧（Shift + 9）: カーソル位置を維持して Y 軸を縮小
* 0 : カーソル位置を維持して拡大
* Ctrl + 0 : カーソル位置を維持して X 軸を拡大
* 右括弧（Shift + 0）: カーソル位置を維持して Y 軸を拡大
* バッククォート : X 軸と Y 軸をウィンドウに合わせる
* 1 : X 軸のみウィンドウに合わせる
* 2 : Y 軸のみウィンドウに合わせる
* k : X の下限範囲をカーソルの X 値に設定
* l : X の上限範囲をカーソルの X 値に設定
* K : Y の下限範囲をカーソルの Y 値に設定
* L : Y の上限範囲をカーソルの Y 値に設定
* g : グリッドのオン／オフを切り替え
* f : 凡例のオン／オフを切り替え

パン
----
* 左矢印 : 左にパン
* 右矢印 : 右にパン
* 上矢印 : 上にパン
* 下矢印 : 下にパン
