Naxis モードは、Ginga 画像ビューアで画像スタックのスライス間を移動する
ためのキー割り当てを有効にします。

モードに入る方法
----------------
* スペース、次に「n」

モードを抜ける方法
------------------
* Esc

モードの標準キー割り当て
------------------------
* スクロール : 現在の軸の前または次のスライスを選択
* Ctrl + スクロール : 前または次の軸を現在の軸として選択
* 左ドラッグ : カーソル位置のウィンドウ幅に対する割合に応じてスライスを選択
* 上下矢印 : 前または次の軸を現在の軸として選択
